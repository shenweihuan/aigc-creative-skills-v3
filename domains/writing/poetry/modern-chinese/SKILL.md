# 现代汉语诗 · 自由的韵律 (Modern Chinese Poetry)

> **"诗不是一种文体。诗是一种观看世界的方式——用感觉而非逻辑，用暗示而非陈述。"**
>
> —— 这不是某个诗人的原话，但这是每一个认真写诗的人最终会抵达的理解。
> 学习现代汉语诗，不是学习"怎么写出押韵的句子"，而是学习**一种用语言创造感受空间的方法**。
> 每一个字都是一扇门。每一行空白都是一次呼吸。

---

## 一、大师档案：为什么是现代汉语诗？

### 它不是"不押韵的古诗"，它是一场语言革命

| 维度 | 信息 |
|------|------|
| 起点 | 1917年胡适《文学改良刍议》——新文学运动的发端 |
| 核心主张 | "我手写我口"——用白话文代替文言文作为诗歌媒介 |
| 关键转折点 | 1918年《新青年》发表第一批白话诗；1920年《尝试集》出版；1978年《今天》杂志创刊（朦胧诗）；1989年海子逝世 |
| 代表诗人（第一代） | 胡适 / 郭沫若 / 徐志摩 / 闻一多 |
| 代表诗人（第二代） | 戴望舒 / 卞之琳 / 何其芳 / 艾青 |
| 代表诗人（第三代/朦胧诗） | 北岛 / 顾城 / 舒婷 / 杨炼 / 多多 |
| 代表诗人（后朦胧/当代） | 海子 / 西川 / 于坚 / 翟永明 / 欧阳江河 |
| 当前状态 | 活跃但分化：学院派 vs 民间派 vs 网络诗歌 vs AI诗歌实验 |

### 为什么它值得你花时间学习？

```
普通人的阅读方式：
  "这首诗写了什么？" → 找主题思想 → 总结中心大意 → 结束 ✓

诗人的阅读方式：
  "这首诗让我感受到了什么？不是'什么意思'，
   是'什么感觉'。那个词为什么放在这里而不是那里？
   那个换行为什么要在这里断？如果不断会怎样？
   那个意象——为什么是月亮不是太阳？为什么是
   一棵树而不是一片森林？这些选择背后有什么？
   我不知道。但我感到了。感到了就够了。
   如果我能说出来，那就是散文，不是诗了。"
   → 反复读 → 每次感受都不同 → 最终你不再需要"理解"，你需要的是"共振"
```

**现代汉语诗的核心贡献不是"白话代替文言"，而是一种全新的语言意识：**

1. **意象的独立性** — 一个意象可以独立存在，不需要依附于完整的叙事或论证
2. **节奏的音乐性** — 自由诗没有格律，但它有节奏——来自呼吸、来自情绪的起伏、来自词语本身的重量
3. **留白的合法性** — 不说出来的比说出来的更重要；读者参与完成意义
4. **日常的诗意** — 最平凡的事物（一把椅子、一个黄昏、一声咳嗽）都可以成为诗的材料

### 三条核心美学脉络

```
脉络A：从胡适到徐志摩——"解放"的美学
  核心：打破旧形式的束缚，让语言回归自然
  特征：口语化、情感直白、追求"真"胜过追求"美"
  代表作：胡适《蝴蝶》、徐志摩《再别康桥》
  局限：有时过于散漫，缺乏内在的凝聚力量

脉络B：从戴望舒到穆旦——"现代性"的美学
  核心：引入西方现代主义的影响，探索语言的更多可能性
  特征：意象复杂化、结构精巧化、智性与感性交织
  代表作：戴望舒《雨巷》、穆旦《诗八首》
  局限：有时过于晦涩，与普通读者产生距离

脉络C：从北岛到海子——"本体"的美学
  核心：诗不再是表达工具，诗本身就是目的
  特征：极端的个人化、语言本身的探险、生命体验的直接转化
  代表作：北岛《回答》、海子《面朝大海，春暖花开》
  局限：高度个人化的表达难以复制和学习（这正是它的价值）
```

## 二、感知训练（Taste Training）

> **品味不是天生的。品味是被训练出来的。**
> 下面4个训练模块，每个都需要你亲自做——不是看，是做。

### 训练1：意象解剖练习（Image Dissection Exercise）

#### 目标

学会把一首诗拆成"意象单元"，理解每个意象的功能和效果。

#### 方法论

```
什么是"意象"？

定义：意象 = 具体的感官形象 + 附着在其上的情感/观念

不是所有"提到的东西"都是意象。
只有当这个东西承载了超出其字面意义的"额外内容"时，它才是意象。

例：海子《面朝大海，春暖花开》中的关键意象：

"面朝大海" → 不是地理方向的选择
           → 是一种姿态：面向无限、面向未知、面向可能性的开放状态
           → 大海 = 广阔、深邃、不可测度、包含一切又拒绝一切

"春暖花开" → 不是气象报告
           → 是一种存在的理想状态：温暖、生机、美好、自然而然地发生
           → 春天 = 希望、新生、循环的开始
           → 花 = 美好的事物的绽放、生命的证明

"喂马、劈柴、周游世界" → 不是旅游计划
                      → 是对"简单幸福生活"的具体想象
                      → 喂马 = 与动物的连接、劳动的节奏
                      → 劈柴 = 身体的参与、原始的能量释放
                      · 周游世界 = 自由、探索、不被固定在一个地方

"关心粮食和蔬菜" → 不是营养建议
                  → 是对最基本的生存之物的珍视
                  → 粮食和蔬菜 = 最朴素的生命必需品
                  → "关心" = 一种态度：不忽视平凡、在平凡中发现神圣

每一个意象都在做同一件事：
  用最具体的东西，指向最抽象的感受。
  这就是好诗的秘密。
```

#### 实操练习

**Step 1**: 选择一首你喜欢的短诗（10-20行）

**Step 2**: 逐行标注所有"意象"（不是每个词都是意象）

标注格式：
```
原文：___________________________
意象：[具体的感官形象] → [它指向的情感/观念]
理由：为什么是这个而不是别的？（简要说明）
```

**示例（以北岛《生活》为例）：

```
原文：网。

意象：[网] → [生活的本质是无法逃脱的结构性束缚]

理由：
  为什么是"网"而不是"墙"？→ 网有空隙，你可以看见外面，
    但你出不去。墙是完全封闭的，网是半开放的——
    这更准确地描述了现代人感受到的那种"有限自由"。

  为什么只有一个字？→ 一个字的诗本身就是一种宣言：
    生活不需要被解释，只需要被命名。
    这个名字就是"网"。够了。
```

**Step 3**: 尝试替换其中一个意象

问自己：
- 替换后的效果变好了还是变差了？为什么？
- 原意象的独特性在哪里？
- 这个意象是否可以被"翻译"成另一种文化语境而不失其力量？

### 训练2：节奏感训练（Rhythm Training）

#### 目标

学会感知和控制自由诗的节奏——虽然没有格律，但有音乐性。

#### 核心概念

```
自由诗的节奏从哪里来？

来源1: 语义节奏（Semantic Rhythm）
  句子的"重量"变化造成节奏。
  重句子（信息密度高、概念抽象）= 慢
  轻句子（信息密度低、具象直接）= 快

  例：
  "我在北京的地铁上想你。" （轻：具体、日常、可想象）
  "关于思念的理论建构。"     （重：抽象、学术、无法想象）
  
  把轻重句子交替排列，就产生了节奏。

来源2: 呼吸节奏（Breath Rhythm）
  朗读时的自然停顿点 = 诗行的分界依据。
  一行诗的长度 ≈ 一次正常呼吸能读完的长度。
  超过这个长度 → 读者会感到"憋"
  远短于这个长度 → 读者会感到"碎"

  例（顾城《一代人》）：
  黑夜给了我黑色的眼睛       （一行，中等长度，一口气读完）
  我却用它寻找光明             （另一行，等长，对称的呼吸节奏）
  
  两行之间的停顿 = 读者消化上一行的时间 + 准备下一行的时间
  这个停顿本身就是节奏的一部分。

来源3: 视觉节奏（Visual Rhythm）
  诗行在页面上的排列方式本身就在传递信息。
  长行 = 展开、叙述、铺陈
  短行 = 收束、强调、顿悟
  孤立的一字/一词 = 极强的聚焦、长时间的凝视
  
  例（极端案例）：
  今天
  我
  想
  你
  
  每个字单独一行 = 每个字都被放大、被孤立、被强制关注
  读者的视线必须垂直移动四次才能读完四个字
  这种"慢"本身就是"想"这个动作的视觉化
```

#### 练习任务

**题目**：为同一个"意思"写出3种不同节奏的版本

**基础意思**："我很想念一个人，但我说不出口。"

**版本A：快速版（短句为主，节奏急促）**
```
想你了。
说不出来。
卡在喉咙里。
像个石头。
咽不下去。
吐不出来。
就这样。
```

**版本B：缓慢版（长句为主，节奏舒缓）**
```
我想念你的那种感觉就像夏天午后的一场雷雨，
你知道它要来了，空气变得粘稠而沉重，
你甚至闻到了雨水落在滚烫沥青路面上的味道，
但你不知道第一滴雨什么时候会落下来，
也不知道这场雨会下多久，
你只是站在窗前等着，手里握着一杯已经凉掉的水。
```

**版本C：变奏版（长短交替，模拟心跳的不规则节奏）**
```
我想你。

不是那种轻飘飘的想，
是沉的。
像一块石头扔进井里——
你听不到它落底的声音，
但你知道它一直在往下坠，
往下坠，
往下坠。

然后有一天你突然发现：
井底原来这么深啊。
```

**分析**：
- 哪个版本最打动你？为什么？
- 每个版本的"情绪速度"有何不同？
- 如果你要表达的不是"想念"而是"愤怒"，你会如何调整节奏？

### 训练3：留白艺术（The Art of Negative Space）

#### 目标

理解"不说"的力量——在现代诗中，留白不是偷懒，是最高的技巧。

#### 核心原则

```
留白的三大类型：

Type A: 意象之间的跳跃（Leap Between Images）
  不解释A和B之间的关系，直接并列。
  读者自己建立连接的过程 = 诗意的产生过程。

  例（顾城《远和近》）：
  你
  一会儿看我
  一会儿看云
  
  我觉得
  你看我时很远
  你看云时很近
  
  分析：
  "你看我时远"和"你看云时近"之间没有"因为...所以..."
  但读者自动补全了因果：因为你对我有防备/隔膜/距离感，
  而你对云没有——云不会伤害你，我会吗？
  这个"补全"的过程就是读者参与创作的过程。
  如果诗人把因果关系写出来，就变成了散文，不是诗了。

Type B: 结尾的突然中断（Abrupt Ending）
  在读者期待"更多"的时候戛然而止。
  余音绕梁的效果来自于"未完成感"。

  例（卞之琳《断章》）：
  你站在桥上看风景，
  看风景的人在楼上看你。
  明月装饰了你的窗子，
  你装饰了别人的梦。
  
  分析：
  四行，完满。但它在哪里"结束"的？
  在"别人的梦"——这是一个永远不会被验证的事实。
  你永远不知道你是否真的装饰了别人的梦。
  诗在这里停下 = 把这个不确定性留给读者。
  如果再加一行解释"所以我们要珍惜相遇"之类的，
  整首诗的美感瞬间归零。

Type C: 未言明的情感（Unstated Emotion）
  从不直接命名情感。
  所有的情感都通过意象和行为来暗示。

  对比：
  ❌ "我很悲伤。"
  ✅ "我把你的照片翻到了背面。"
  
  后者没有出现"悲伤"这个词，
  但"翻到背面"这个动作包含了：
  - 不忍心看（太痛苦）
  - 又舍不得丢掉（还爱着）
  - 试图控制自己的情绪（理性地把它翻过去）
  
  一个动作 > 一百个形容词
```

#### 练习：改写练习——删除所有直接的情感表达

**原文本**（充满情感词汇的"坏诗"）：
```
我今天非常难过，因为我想起了以前的事情。
那些回忆让我觉得很痛苦，但又有点温暖。
我不知道该怎么办，心里乱糟糟的。
我希望有人能理解我，但我知道没人能真正理解。
```

**任务**：重写这段文字，要求：
1. 删除所有情感词汇（难过、痛苦、温暖、乱糟糟、希望、理解...）
2. 只用具体的动作、物体、环境细节
3. 字数不超过原文本的50%
4. 保留（甚至增强）原有的情绪感染力

**参考答案之一**：
```
下午三点。
阳光落在桌子上那杯冷掉的茶上。
我盯着茶叶。
它们都沉在底部了。
窗户开着。
风进来，窗帘动了一下。
我没动。
```

**自检**：
- 你的改写是否仍然传达了"难过"？
- 如果传达了，你是通过什么做到的？
- 改写后的文字是否有更多的"解读空间"？

### 训练4：AI写诗陷阱检测（AI Poetry Pitfall Detection）

#### 目标

识别AI生成诗歌中最常见的缺陷模式，学会避免或修正它们。

#### AI写诗的10大典型缺陷

```
缺陷 #1: 过度抒情（Excessive Lyricalism）
  表现：大量的感叹号、强烈的情感形容词堆叠、
        "啊！""哦！"等语气词泛滥
  例："啊！我的心在燃烧！多么痛苦的思念！
      像火焰一样灼烧着我的灵魂！"
  原因：AI对"诗"的理解停留在"抒情=诗"的浅层
  修正：禁止使用感叹号；限制情感形容词数量（全诗≤3个）；
        用意象替代情感命名

缺陷 #2: 押韵强迫症（Rhyme Obsession）
  表现：每句话尾都押韵，为了押韵牺牲语义自然性
  例："今天的天气真好啊（a），鸟儿在枝头叫哇叫哇（a），
      我的快乐无法形容啊（a），就像盛开的花朵呀（a）"
  原因：AI的训练数据中大量儿歌/顺口溜影响了"诗"的概念
  修正：明确告知"不要押韵"；或者"如果要押韵，必须是自然的、
        不露痕迹的韵"；自由诗本来就不需要押韵

缺陷 #3: 抽象概念堆砌（Abstract Concept Piling）
  表现：整首诗全是抽象名词，没有一个具体意象
  例："时间的流逝带来了生命的感悟，
      灵魂在宇宙中寻找永恒的意义，
      爱与恨交织成复杂的情感网络..."
  原因：AI倾向于"概括"而非"展示"
  修正：执行"意象转换"——每个抽象概念必须找到至少一个
        具体的感官对应物（时间→时钟/日历/皱纹；
        灵魂→胸口/眼睛/呼吸；爱→手心的温度/失眠的夜）

缺陷 #4: 说教倾向（Didactic Tendency）
  表现：诗的最后一定要总结出一个"道理"或"哲理"
  例：（前面写得还行）"...所以我们应该珍惜当下，
      把每一天都当作最后一天来活。"
  原因：AI认为"有意义"= "有教训"
  修正：明确禁止"总结""升华""寓意""告诉我们"；
        好诗不给出答案，好诗提出问题

缺陷 #5: 陈词滥调（Cliché Overload）
  表现：月光代表思念、落叶代表悲伤、春天代表希望...
        所有意象都是被用烂了的符号
  例："月亮弯弯照我心，思乡之情油然而生..."
  原因：AI的概率模型倾向于输出高频组合（最常见的搭配）
  修正：提供"反套路"指令——"不要用月亮表示思念，
        找一个新的意象"；建立个人的"新鲜意象库"

缺陷 #6: 结构过于规整（Over-Structured）
  表现：每段行数相同、每行字数相近、
        结构完美对称但缺乏有机感
  原因：AI对"秩序"的偏好
  修正：鼓励不规则——"有些行很长有些行很短"、
        "段落长度不均匀"、"允许有一行特别突出或特别短"

缺陷 #7: 缺乏个人声音（Absence of Personal Voice）
  表现：诗可以是任何人的，看不出作者的身份/经历/独特视角
  原因：AI没有"自我"，它输出的是统计平均值
  修正：在提示词中注入具体的人格设定——
        "你是一个XX岁的XX职业的人，正在经历XX..."

缺陷 #8: 意象密度失控（Image Density Out of Control）
  表现 A：太少（整首诗只有1-2个意象，其余全是空话）
  表现 B：太多（每行塞满意象，像超市货架一样拥挤）
  原因：对"多少合适"缺乏判断
  修正：设定目标密度——"每3-5行至少有一个新鲜意象"、
        "同时出现的意象不超过3个，给每个意象足够的空间"

缺陷 #9: 时间感的缺失（Temporal Flatness）
  表现：诗发生在"无时间"的真空里，看不出是早晨还是夜晚、
        是回忆还是当下、是瞬间还是永恒
  原因：AI默认不处理时间维度
  修正：强制加入时间锚点——"现在是几点？""这件事发生在什么时候？"
        "这个意象属于哪个季节/时段？"

缺陷 #10: 假深刻（Pseudo-Profoundity）
  表现：用看似深奥实则空洞的语言制造"有深度"的假象
  例："在虚无的边缘，存在主义的钟声敲响了
      形而上学的迷雾笼罩着本体论的荒原..."
  原因：AI误以为"大词"="深刻"
  修正：禁止使用超过3个字的哲学术语；
        要求"用小学生能懂的话写出深刻的感受"
```

#### 练习：AI诗歌诊断室

**任务**：让AI写一首诗（任何主题），然后用上述10个缺陷清单进行诊断

**操作步骤**：
1. 给AI一个简单的写诗指令（如"写一首关于下雨的诗"）
2. 获得AI的输出
3. 逐条对照10个缺陷清单，标记哪些出现了
4. 写一份"诊断报告"：
   - 出现了哪些缺陷？（列出编号和具体位置）
   - 最严重的问题是哪个？
   - 如何修正？（给出具体的修改建议）
5. 让AI根据你的诊断报告重新写一版
6. 对比两版，评估改进幅度

## 三、AI提示词公式（Modern Poetry Prompt Formula）

### 通用公式结构

```
[人格设定] + [主题/触发点] + [风格约束] + [禁忌清单] + [技术参数]
= 完整的现代诗生成提示词
```

### 公式详解

#### Component 1: 人格设定（Persona）— 谁在写这首诗

```markdown
这是最重要的差异化组件！

❌ 差的设定：
"你是一个诗人" → 太泛，AI不知道该调用哪种"诗人"模板

✅ 好的设定（具体、有身份、有经历）：

【设定A：都市失眠者】
"你是一个28岁的上班族，住在北京的出租屋里，
凌晨三点还没睡着。窗外有路灯的光透过窗帘缝隙照进来。
你最近刚结束一段感情，但你不打算写关于'失恋'的诗——
你想写的是'失眠时看到的那些平时注意不到的东西'。"

【设定B：返乡的异乡人】
"你是一个35岁的人，在大城市工作了十年之后第一次回老家。
老家的房子还在，但邻居们都不认识了。你想写一首关于
'回不去的家'的诗，但不要用'乡愁''怀旧'这种词——
你要写的是你在老房子里看到的那些具体的、变了的东西。"

【设定C：孩子的眼睛】
"你是一个7岁的小孩。你不懂什么叫'诗'。
你只是把你今天看到的最奇怪/最美/最害怕的东西
用你说的话写下来。不要用大人说话的方式。
用孩子的方式——短句、跳跃、不加解释。"
```

#### Component 2: 主题/触发点（Topic/Trigger）— 写什么

```markdown
原则：给"触发点"而不是给"主题"

❌ 差：
"写一首关于爱情的诗"
→ 太宽泛，AI会输出最通用的爱情诗模板

✅ 好（给具体的触发场景）：

【触发A】
"触发场景：你在地铁上看到一个穿蓝色外套的人，
背影很像你已经失去联系三年的朋友。
你没有上前打招呼。车到站了，那个人下车了。
你看着车门关上。"

【触发B】
"触发场景：你奶奶的老花镜放在窗台上，
灰尘落在镜片上。阳光穿过镜片，
在桌面上投下一个小小的彩虹光斑。
你奶奶已经去世两年了。"

【触发C】
"触发场景：你加班到晚上十一点走出办公楼，
抬头看到一轮很大的月亮挂在两栋高楼之间。
你突然意识到这是你这周第一次抬头看天空。"
```

#### Component 3: 风格约束（Style Constraints）— 怎么写

```markdown
=== 语言约束 ===
| 约束 | 说明 |
|------|------|
| 口语化优先 | 使用日常对话中的语言，避免书面语/文艺腔 |
| 短句为主 | 平均每行不超过15个汉字 |
| 具体意象 | 每个抽象概念必须转化为具体形象 |
| 不用成语 | 成语是"预制的感觉"，诗需要"现场制作的感觉" |

=== 结构约束 ===
| 约束 | 说明 |
|------|------|
| 自由分段 | 不要规定段落数量或每段行数 |
| 允许单字成行 | 如果一个字足够有力，就让它独占一行 |
| 长短交错 | 有意识地创造节奏变化 |

=== 节奏约束 ===
| 约束 | 说明 |
|------|------|
| 可朗读性 | 写完后必须能够不出声地默读流畅 |
| 呼吸点 | 换行处应该是自然的呼吸停顿点 |
| 速度变化 | 全诗应该有快慢变化，不能匀速 |
```

#### Component 4: 禁忌清单（Prohibition List）— 绝对不要做的

```markdown
🔴 绝对禁止：
  - 不要使用感叹号（!）
  - 不要使用"啊""哦""唉"等语气词
  - 不要总结、升华、讲道理
  - 不要使用"仿佛""好像""似乎"等推测性词语（除非是真的视觉类比）
  - 不要使用"的"字频率超过每行1次
  - 不要押韵（除非是极其自然的偶发性押韵）

🟡 尽量避免：
  - 不要使用"月亮=思念""落叶=悲伤""春天=希望"等套路意象
  - 不要使用超过10个字的哲学/抽象名词
  - 不要使用排比句式（"有的...有的...有的..."）
  - 不要在结尾处"点题"或"呼应开头"
```

#### Component 5: 技术参数（Platform-Specific）

```markdown
各AI平台配置建议：

ChatGPT (GPT-4o):
  - 最佳策略：先给2-3个好范例（few-shot），再给任务
  - Temperature: 0.8-0.9（诗歌需要一定的随机性和创造性）
  - System Prompt 强调："你不是在写'看起来像诗的文字'，
    你是在用语言创造一种体验。"

Claude (Claude 3.5 Sonnet):
  - Claude 对中文诗歌的理解质量较高
  - 特别适合处理"含蓄""留白""暗示"类的需求
  - 建议：给它看一首你喜欢的中文诗作为风格参考

Gemini:
  - 倾向于输出较长、较"完整"的作品
  - 需要更严格地限制篇幅和结构
  - 建议：明确指定行数范围（如"10-20行"）

国产大模型 (文心/通义/智谱):
  - 对现代诗的"自由度"理解有时不足（容易写成顺口溜）
  - 建议：增加"这不是顺口溜/打油诗/儿歌"的强约束
  - 提供范例的效果比纯指令更好
```

### 完整示例

#### 示例1：深夜便利店（原创主题）

```
Prompt:
你是一个26岁的女生，在深圳的一家互联网公司做运营。
今天是周五，但你加班到了晚上十一点半。
回家的路上你去楼下的24小时便利店买了一盒牛奶。
便利店的灯光很亮，亮得不真实。
店员在玩手机，收银台旁边有一只猫在睡觉。
你想写一首关于"这个时刻"的诗。

风格要求：
- 用最平实的语言，不要文艺腔
- 不要说"累""孤独""想家"这类词
- 关注你能看到、听到、触到的具体事物
- 每行不超过12个字
- 可以有一两个字单独占一行
- 结尾不要总结，停在某个画面或感觉上
- 总共15-25行

禁忌：
❌ 感叹号 ❌ "啊""哦" ❌ 押韵 ❌ 成语
❌ "仿佛""好像"（除非是真实的视觉描写）

#### 示例2：旧物（致敬风格）

```
Prompt:
你是一个45岁的人。周末你在整理父母的老房子，
准备搬家的东西。
你在一个抽屉里找到了一本相册，
是你小时候的照片，但你完全不记得拍这些照片时候的事了。
照片里的你笑得很开心。
你现在不太记得上次那么开心是什么时候了。

请以这本相册为核心意象写一首诗。

要求：
- 不要写"时光流逝""童年已逝"这种大词
- 选择照片中的一个具体细节来展开（比如背景里的某样东西、
  你的某个表情、照片的物理状态如卷角/泛黄）
- 用"现在的你"和"照片里的你"做一个对话（不用引号，

#### 示例3：城市观察（练习型）

```
Prompt:
你是一个诗人。你的方法是：走到街上，
随机选择一个普通人（不与他/她互动），
观察他/她30秒，然后基于这30秒的观察写一首短诗。

今天的观察对象：
一个中年男人，穿着有点皱的白衬衫，
提着一个塑料袋（里面似乎装着蔬菜），
站在红绿灯路口等绿灯。
他在看手机，眉头微皱。
绿灯亮了，他收起手机，快步走过马路。

基于以上观察写一首诗（8-15行）。

原则：
- 不要编造你没有观察到的东西
- 不要猜测他是谁、在想什么、要去哪里
- 只写你看到的 + 这些画面给你的直接感受
- 用最少的字达到最大的感受密度
```

## 四、评价标准（How to Judge Quality）

### 现代诗五维自检清单

| 维度 | 1分（差） | 3分（及格） | 5分（优秀） | 自检方法 |
|------|----------|-----------|------------|---------|
| **意象质量** | 全是抽象概念或陈词滥调（月亮=思念、秋天=悲伤） | 有具体意象但不够新鲜，或意象与情感连接牵强 | 意象独特、具体、有感官冲击力；每个意象都精准地指向一种难以命名的感受 | 盖住诗的下半部分，仅凭上半部分的意象能否猜到大致情绪？如果能且准确=好 |
| **节奏感** | 匀速、单调，像在读说明书 | 有一定起伏但变化不够明显或不够有意 | 有明确的快慢变化、呼吸感、某些地方的"意外停顿"或"加速"服务于情绪 | 默读一遍——有没有哪个地方让你不由自主地停长了？有没有哪个地方让你加快了？ |
| **留白程度** | 一切都说尽了，读者没有参与空间 | 有一些未言明之处但整体还是太"满" | 精准的控制——说的刚好够让你感受到，剩下的你自己补 | 读完后你是否有"还想再多想一想"的感觉？如果有=留白成功 |
| **语言质感** | 充满"诗歌套话"（由于/于是/仿佛/宛如）、书面腔、翻译腔 | 语言通顺但缺乏个人特色，像是"任何人都能写出的诗" | 有独特的语言指纹——词汇选择、句法习惯、观察角度都是"这一个诗人"的 | 如果遮住作者名，你能认出这是谁的风格吗？（对于AI生成的诗：能否看出是"哪个人格设定"写的？） |
| **情感真实性** | 情感虚假或过度夸张，像在表演感情 | 有真情但表达方式略显刻意 | 情感像是从生活中直接提取的——不加工、不美化、不夸大，但因为真实所以动人 | 读的时候你有生理反应吗？（心跳、呼吸、眼眶、皮肤）——诗的好坏的终极检测标准是身体，不是大脑 |

### 快速评分流程

```
生成诗歌后，按以下步骤快速自检（3分钟）：

Step 1: 第一遍默读（30秒）
  → 有没有哪个地方让你停下来？（好的停顿=回味；坏的停顿=困惑）
  → 有没有哪个地方你想跳过？（想跳过=这部分无效）
  → 整体感受用一个词形容：______________

Step 2: 意象清点（30秒）
  → 数一数有多少个独立的、具体的意象
  → < 3个 = 意象维度 ≤ 2分（太稀疏）
  → 3-6个 = 3-4分
  → > 6个但每个都有足够的空间 = 5分
  → > 6个且拥挤 = 3分（过载）

Step 3: 情感词汇扫描（30秒）
  → 用Ctrl+F搜索以下词：
    难过/开心/悲伤/快乐/孤独/思念/痛苦/幸福/感动/
    爱/恨/美/丑/永恒/瞬间/灵魂/心灵/梦想...
  → 出现数量：
    0个 = 可能很好（完全靠意象传达）
    1-2个 = 可接受（如果位置巧妙）
    3-5个 = 需要修改
    > 5个 = 严重依赖情感标签，≤ 2分

Step 4: "翻译测试"（60秒）
  → 尝试把这首诗"翻译"成散文（把所有分行去掉，
    连成一段普通文字）
  → 如果翻译后的散文仍然有诗意 = 原诗的分行和节奏
    确实起到了作用（加分项）
  → 如果翻译后变成了一堆普通的陈述句 =
    原诗的形式大于内容（扣分项）

Step 5: 综合判断
  → 平均分 ≥ 4.0：优秀
  → 3.0-3.9：良好，针对性修改
  → < 3.0：需要重新构思，不要在现有版本上修补
```

## 五、常见陷阱与修正（Pitfalls & Fixes）

| # | 陷阱 | 典型表现 | 原因 | 修正方案 |
|---|------|---------|------|---------|
| P1 | **变成顺口溜/打油诗】 | 每句都押韵、朗朗上口但没有深度、像儿歌 | AI对"诗"= "押韵的文字"的刻板印象 | 明确禁止押韵；给出"不押韵的好诗"范例；强调"自由诗不需要押韵" |
| P2 | **过度抒情/滥情】 | 大量感叹号、"啊！""哦！"、浓得化不开的情感形容词 | AI认为"诗"= "强烈情感的宣泄" | 设定情感词汇配额（全诗≤3个）；用"情感禁令列表" |
| P3 | **说教/升华】 | 结尾一定要总结一个道理，"这让我明白了..." | AI认为好文章必须有"教育意义" | 明确禁止结尾升华；要求"停在画面/感觉上，不要解释" |
| P4 | **意象陈旧】 | 月亮=思念、落叶=悲伤、大海=辽阔、花朵=青春 | AI概率模型倾向于高频组合（最"安全"的选择） | 提供"反套路"指令；建立"新鲜意象库"；要求"找一个从未被用于此情感的意象" |
| P5 | **结构过于整齐】 | 每段4行、每行字数相近、像填空题 | AI对"秩序"的偏好 | 故意要求"不规则结构"；"有些行很长有些很短" |
| P6 | **缺少时间锚定】 | 发生在真空里，看不出何时何地 | AI默认不处理时空坐标 | 强制要求包含至少一个时间指示词（"三点""清晨""去年"）和一个空间指示词（"地铁站""厨房""那条路"） |
| P7 | **AI腔残留】 | 即使设定了诗人人格，仍出现"让我们...""这象征着...""正如..." | AI的"助手"角色惯性 | 使用更强的角色扮演指令；追加"清除AI腔"的后处理步骤 |

### 陷阱深度解析：P4 意象陈旧

```
问题诊断：

  陈旧意象库（AI默认输出）及其"毒性"：

  月亮 → 思念/孤独/远方
    🤢 毒性等级：极高（几乎每次都会出现）
    💊 解药：换成更具体的月亮——"凌晨四点的月亮"
         （暗示失眠/特定时间/非浪漫化的月亮）

  秋天 → 悲伤/凋零/终结
    🤢 毒性等级：高
    💊 解药：换成秋天的某个具体侧面——
         "梧桐树掉下来的第三片叶子落在我的伞上但没有声音"

  大海 → 辽阔/自由/未知
    🤢 毒性等级：中高
    💊 解药：换成海的某种异常状态——
         "退潮后的海滩上，一只螃蟹横着走了很久"

  花朵 → 美好/青春/爱情
    🤢 毒性等级：中
    💊 解药：换成花的非审美状态——
         "路边的野花开得太用力了，花瓣边缘有点焦"

  雨 → 忧郁/清洗/泪水
    🤢 毒性等级：中
    💊 解药：换成雨的非浪漫化细节——
         "雨打在空调外机上，声音像有人在敲门但没有人开"

  新鲜意象生成公式：
    [常见物品] + [异常状态] + [精确观察] = 新鲜意象
    
    例：
    常见物品：一把椅子
    异常状态：空着的、但在上面还有余温
    精确观察：坐垫上有一个浅浅的凹痕
    结果："椅子上的凹痕还没有弹回来"
    → 暗示：刚刚有人坐在这里 / 那个人走了 /
       时间很短（凹痕还在）/ 一种微妙的缺席感
```

## 六、跨域迁移（Cross-Domain Transfer）

> **品味的最高级应用：把在一个领域学到的东西用到另一个完全不同的领域。**

### 迁移地图

| 目标领域 | 迁移什么？ | 怎么迁移？ | 示例 |
|---------|-----------|-----------|------|
| **歌词创作** | 意象密度控制 + 节奏感 | 歌词本质上是有旋律的诗；现代诗的意象方法可以直接应用于歌词 | 李宗盛/罗大佑/陈奕迅的歌词有明显的"诗性"——他们都是先写好诗再谱曲或反之 |
| **广告文案** | 留白艺术 + 以少胜多 | 最好的广告文案像一首短诗——不说透、留余地、让读者自己得出结论 | "Think Different"（苹果）= 一行现代诗；"钻石恒久远"（De Beers）= 四行现代诗 |
| **短视频脚本** | 视觉意象思维 | 把诗的"文字意象"转化为"镜头意象"——每个镜头都是一个意象单元 | 抖音/小红书上最高赞的内容往往遵循"意象并置+意外结尾"的诗性结构 |
| **小说/散文** | 语言的精确性和节奏感 | 小说中的"诗性段落"往往是全文最动人的部分；用写诗的标准要求散文的关键段落 | 鲁迅《野草》= 散文诗的典范；张爱玲的比喻系统接近诗歌意象 |
| **产品设计** | "少即是多"的理念 | 诗的留白 = 设计的负空间；诗的意象聚焦 = 设计的功能聚焦 | Apple的设计哲学与极简主义诗歌共享同一种美学基因 |

### 具体迁移练习：诗歌式广告文案

```
原版广告（某咖啡品牌）：
"☕ XX咖啡，精选阿拉比卡豆，经过精心烘焙，
带来醇厚口感和浓郁香气。无论是忙碌的工作日清晨，
还是悠闲的周末午后，一杯XX咖啡都能让您享受
片刻的宁静与美好。选择XX咖啡，选择品质生活。"

问题：说了太多（阿拉比卡、精心烘焙、醇厚、浓郁、
忙碌、悠闲、宁静、美好、品质），消费者记不住任何一句。

现代诗版（迁移留白+意象+克制）：
"七点的电梯。
十二点的会议室。
六点的末班车。

中间的那杯，
还烫着。"

分析：
- 原版：80+字，信息密度低，全是抽象赞美
- 诗版：16字，信息密度极高，每个短语都是具体场景
- 原版试图说服（"让您享受""选择品质"）
- 诗版只呈现（电梯/会议室/末班车/中间/烫着）
- 读者脑补：这个人很忙、咖啡是他唯一的喘息机会、
  这杯咖啡对他很重要、我也想要这样的时刻
- "还烫着"两个字的分量 > 原版全部80个字
```

```
Level 1 完成（你现在这里）：

    - 第22天回看，找出你自己的"语言指纹"
  → 做1次"AI诗歌诊所"项目：
    收集10首AI生成的诗（不同平台的、不同提示词的），
    用10大缺陷清单逐一诊断，写出改进方案

Level 3 融合（建议3-6个月）：
  → 开始跨域迁移尝试（至少2个不同领域）
  → 尝试将现代诗与其他技能融合

         现代诗 × 吉卜力世界观 = ?）
  → 建立"个人的意象库"：
    收集50个你认为"新鲜"的意象，
    分类整理，定期更新
  → 能够在30秒内判断一首诗"够不够好"以及"差在哪里"

Level 4 创新（长期目标）：
  → 发展出个人风格的现代诗（无论是否借助AI）
  → 理解现代诗的局限（是的，它有局限）：
    - 受众越来越小众（"诗"在公共话语中的地位下降）
    - 过度精英化/学院化的风险
    - 与流行文化的割裂
  → 回答这个问题："AI时代，诗的价值是什么？
     如果AI能写出'看起来像诗'的东西，
     人类写诗的意义在哪里？"
  → 这个问题的答案就是你自己的创新起点
```
