# 古典诗词 · 意境之美 (Classical Chinese Aesthetics)

> **"词以境界为最上。有境界则自成高格，自有名句。"**
>
> —— 王国维《人间词话》开篇第一句。
> 学习古典诗词，不是学习"怎么写出押韵的文言文"，而是学习**一种用极少的字创造无限空间的方法**。
> 每一个字都是一座山。每一处留白都是一片海。

---

## 一、大师档案：为什么是古典诗词？

### 它不是"老古董"，它是中国美学的最高成就

| 维度 | 信息 |
|------|------|
| 起点 | 《诗经》（约公元前11世纪-前6世纪）——中国诗歌的源头 |
| 高峰期 | 唐诗（618-907）+ 宋词（960-1279）——两个不可逾越的高峰 |
| 核心载体 | 诗（五言/七言/古体/近体）+ 词（长短句）+ 曲（杂剧散曲） |
| 关键概念 | 意境 / 留白 / 对仗 / 格律 / 兴观群怨 / 赋比兴 |
| 代表诗人（唐） | 李白 / 杜甫 / 王维 / 白居易 / 李商隐 |
| 代表词人（宋） | 苏轼 / 辛弃疾 / 李清照 / 柳永 / 周邦彦 |
| 美学理论家 | 司空图（《二十四诗品》）/ 严羽（《沧浪诗话》）/ 王国维（《人间词话》） |

### 为什么它值得你花时间学习？

```
普通人的阅读方式：
  "这首古诗写了什么？" → 翻译成白话文 → 理解意思 → 结束 ✓

古典美学的阅读方式：
  "这首诗创造了一个什么样的世界？
   这个世界里有光吗？有声音吗？有温度吗？
   诗人在这个世界的什么位置？他在看什么？
   他没有写的东西比他写了的更重要——
   那些空白里藏着什么？
   
   '明月松间照，清泉石上流。'
   不只是说'月亮照在树林里，泉水流在石头上'。
   是说：在这个世界上，
   光是安静的（照，不是射/不是洒），
   水是清澈的（清泉）、是可见其底的（石上流=水浅石现）、
   是有声音的（流=潺潺之声虽未明写但可闻）。
   而且这一切发生在夜晚（明月）的山林间（松间）。
   你不在那里。但你通过这十个字，就在那里了。
   
   这就是'意境'——不是描写一个地方，
   是把你放到那个地方去。"
   → 反复读 → 每次都在那里 → 最终你带着那里的气息回到现实
```

**古典诗词的核心贡献不是形式（格律/对仗），而是一种时空折叠技术：**

1. **意境（Yìjìng）** — 用有限的文字创造无限的感受空间；读者不是在"读"诗，是在"进入"一个世界
2. **留白（Liúbái）** — 不说的比说的更重要；空白处有最大的信息量
3. **对仗（Duìzhàng）** — 不是文字游戏，而是通过对称制造张力和平衡的美学装置
4. **格律（Gélǜ）** — 限制激发创造力；在最严格的规则下寻找最大的自由

### 四大核心美学概念

#### 概念1：意境（Yìjìng —— Artistic Conception / Idealmood）

```
王国维的定义：
  "境非独谓景物也。喜怒哀乐，亦人心中之一境。"
  
  翻译：意境不只是风景描写。
  喜怒哀乐这些情感本身也是一种"境"。

三层结构：

Layer 1: 景（Objective Image）
  诗中描写的客观物象
  例："大漠孤烟直，长河落日圆"中的
      大漠、孤烟、长河、落日

Layer 2: 情（Subjective Emotion）
  诗人注入的主观情感
  例：王维写下这两句时的心境——
      孤独但辽阔、渺小但不卑微、
      在极致的荒凉中发现极致的美

Layer 3: 意（Synthesized Meaning）
  景与情融合后产生的、超越两者之和的"第三种东西"
  这就是"意境"本身——无法被完全分解为景或情的
  那种让你读了之后"心里有什么东西被触动了但又说不清"的感觉
  
  例："大漠孤烟直，长河落日圆"的意境 =
     不只是"壮丽的边塞风光"（太浅）
     也不只是"诗人的孤独感"（太窄）
     而是：人类在浩瀚自然面前的存在感——
     渺小但不悲哀、孤独但有尊严、
     时间（长河=时间的流动）与空间（大漠=空间的广度）
     在这一刻交汇，而你作为一个读者，
     也被拉进了这个交汇点。

关键洞察：
  好诗的意境 = 让你忘记你在"读诗"。
  你以为你在看一幅画、听一段音乐、站在某个地方。
  等你回过神来，你才发现：那只是二十个汉字。
  但那二十个汉字刚刚带你去了另一个世界。
```

#### 概念2：留白（Liúbái —— Negative Space / The Unsaid）

```
中国画的最高原则也是诗的最高原则：
  计白当黑——空白的地方也是画面的一部分。

在诗中，留白的运作方式：

类型A: 景物的省略（Omission of Scenery）
  只写局部，让读者补全整体。
  
  例："枯藤老树昏鸦"
  → 只写了三种事物，但读者自动脑补了一幅完整的秋日黄昏图：
    枯藤缠绕的老树上停着一只乌鸦，
    天色已暗，风大概有点凉...
  → 如果王实甫把所有细节都写出来，这首曲就死了。

类型B: 情感的暗示（Implication of Emotion）
  从不直接命名情感，只写引发情感的外部迹象。
  
  例："执手相看泪眼，竟无语凝噎"（柳永《雨霖铃》）
  → 没有"悲伤""不舍""痛苦"这些词
  → 但"执手""相看""泪眼""无语""凝噎"六个动作/状态
    组合起来 = 最深的不舍和最重的悲伤
  → 如果加上"我很舍不得离开你"，整首词的意境归零。

类型C: 时间的跳跃（Temporal Leap）
  省略过程，直接呈现结果或跨越时间段。
  
  例："少小离家老大回"（贺知章《回乡偶书》）
  → 五十年的人生历程压缩在一行五个字里
  → "少小"到"老大"之间发生了什么？
    读者自己填充：成长、奋斗、漂泊、衰老、回忆...
  → 这五十年越是空白，读者的想象就越丰富

类型D: 因果的隐藏（Hidden Causality）
  不解释为什么，只呈现结果。
  
  例："感时花溅泪，恨别鸟惊心"（杜甫《春望》）
  → 为什么花会让诗人流泪？为什么鸟会让他心惊？
    不解释。
  → 读者自行推断：因为内心极度悲痛，
    所以看到美好的花反而更难过（对比效应）；
    因为神经高度紧张/敏感，
    所以鸟的正常鸣叫也像惊吓。
  → 这种推断的过程 = 读者深度参与创作的过程
```

#### 概念3：对仗（Duìzhàng —— Parallelism / Antithesis）

```
对仗不是"文字游戏"。对仗是一种认知装置。

基本规则：
  字数相等、词性相同、平仄相对、意义相关（或相反）

但对仗的美学功能远超技术层面：

功能1: 制造张力（Creating Tension）
  通过并置对立面，产生"两极之间的振动"。
  
  例："无边落木萧萧下，不尽长江滚滚来"（杜甫）
       无边 vs 不尽（空间的无限 vs 时间的无限）
       落木 vs 长江（生命的消逝 vs 时间的永恒）
       萧萧下 vs 滚滚来（向下的消逝 vs 向前的永恒）
  → 两行诗之间形成了一个巨大的张力场，
    读者被夹在"有限的生命"和"无限的时间"之间，
    这种夹击感 = 对仗的力量。

功能2: 创造平衡（Creating Balance）
  通过对称结构带来视觉和心理的稳定感。
  
  例："两个黄鹂鸣翠柳，一行白鹭上青天"（杜甫）
       两个 vs 一行（数量对）
       黄鹂 vs 白鹭（颜色+鸟类对）
       鸣 vs 上（动词对）
       翠柳 vs 青天（颜色+景物对）
  → 完美的平衡 = 春日的和谐美好
  → 但如果仔细看：黄鹂在低处（柳树），白鹭在高处（青天）
    → 平衡中有层次，对称中有纵深

功能3: 强化记忆（Enhancing Memory）
  对仗的结构天然容易被记忆——因为大脑喜欢模式。
  许多千古名句之所以流传，部分原因在于其对仗的"粘性"。
  
  例："海内存知己，天涯若比邻"（王勃）
       海内 vs 天涯（空间的对立）
       存知己 vs 若比邻（关系的超越）
  → 十个字，记一辈子。
```

#### 概念4：格律（Gélǜ —— Metric Pattern / Prosody）

```
格律的本质：在限制中寻找自由。

为什么需要格律？
  不是为了"难"，是为了"聚焦"。
  格律像一个框架，强制诗人把每一个字都用到极致——
  因为空间有限，所以每个位置都必须是最优选择。

近体诗（唐代定型）的基本格律系统：

=== 五言律诗（每句5字，共8句） ===
平仄格式（简化版）：
  （仄）仄（平）平仄，（平）平（仄）仄平
  （平）平（仄）仄仄，（仄）仄（平）平仄
  ...后四句与前四句镜像对称...

关键约束：
  - 每句字数固定（5字或7字）
  - 平仄声调交替有严格规定
  - 第2、4、6、8句必须押韵（首句可选）
  - 中间两联（第3-4句、第5-6句）必须对仗

=== 词（Song Lyric）的格律 ===
  - 句子长短不一（故称"长短句"）
  - 每个词牌（如《菩萨蛮》《蝶恋花》《水调歌头》）有固定的字数和平仄格式
  - 押韵的位置因词牌而异
  - 对仗要求因词牌而异（有些严格要求，有些完全不要求）

格律对AI创作的启示：
  AI生成古典诗词时，格律是一个硬性约束——
  不是"最好遵守"，而是"必须遵守"。
  不合律的"古风"只是穿了古装的白话文，
  不是真正的古典诗词。
```

## 二、感知训练（Taste Training）

> **品味不是天生的。品味是被训练出来的。**
> 下面4个训练模块，每个都需要你亲自做——不是看，是做。

### 训练1：意境还原练习（Yìjìng Reconstruction Exercise）

#### 目标

学会把一首古典诗词"翻译"成一个完整的世界——不是翻译意思，是翻译感受。

#### 方法论

```
操作步骤：

Step 1: 选一首短诗（绝句或小令最佳，20-30字）
        推荐：
        - 王维《鹿柴》：空山不见人，但闻人语响。
          返景入深林，复照青苔上。
        - 柳宗元《江雪》：千山鸟飞绝，万径人踪灭。
          孤舟蓑笠翁，独钓寒江雪。
        - 马致远《天净沙·秋思》：枯藤老树昏鸦，
          小桥流水人家，古道西风瘦马。

Step 2: 逐字拆解意象
        为每个意象建立一张"卡片"：

        【意象卡片模板】
        意象：______
        字面含义：______
        感官通道：（视觉/听觉/触觉/嗅觉/味觉）
        属性细节：（颜色/形状/声音/温度/质地...）
        情感联想：（这个意象通常唤起什么感觉？）
        在本诗中的特殊用法：（是否有反常/创新之处？）

Step 3: 用现代汉语"重建"这个世界
        要求：
        - 不要翻译成白话散文（那是另一回事）
        - 写一段100-200字的"场景描述"
        - 描述的重点是：如果你真的站在那个世界里，
          你的五官会接收到什么信息？
        
        示例（以王维《鹿柴》为例）：
        
        "这是一座山。不是旅游景点的山，
         是那种连砍柴人都不愿意来的深山。
         安静到你能听到自己的心跳。
         突然——有人说话的声音。
         你看不到人，但声音在那里，
         从某个方向传来，又消失了。
         
         太阳快下山了。一束光——
         不知道从哪个缝隙里钻进来的——
         斜斜地照进树林深处。
         光落在青苔上。
         那片青苔平时见不到阳光，
         此刻被照亮了，绿得发亮。
         光在移动。慢慢地。
         过一会儿，这片青苔又会回到黑暗里。
         但现在，它是全森林最亮的东西。"

Step 4: 对比原诗和你的重建
        - 你的描述是否"多"了原诗没有的信息？（如果是，哪些是合理的推演？哪些是过度解读？）
        - 原诗用了20个字，你用了150个字——
          多出来的130个字是"必要的"还是"多余的"？
        - 原诗的哪几个字是你无论如何也"翻译"不出来的？
          （那些字就是"诗意"所在——不可翻译的核心）
```

### 训练2：留白侦探练习（Negative Space Detective）

#### 目标

学会识别一首诗中"没写的东西"——那些空白处藏着的才是真正的信息。

#### 方法论

```
核心原理：
  一首好诗的信息分布：
  
  ┌─────────────────────────────┐
  │  写出来的（文字表面）  30%  │ ← 显性信息
  ├─────────────────────────────┤
  │  暗示出来的（上下文推导）40% │ ← 隐性信息
  ├─────────────────────────────┤
  │  完全空白的（读者自填）  30% │ ← 创造性信息
  └─────────────────────────────┘
  
  差诗的问题：100%的信息都写在表面（甚至不到——
             因为表面的信息往往是陈词滥调）
  好诗的魔法：用30%的文字触发100%的感受

实操方法：

Step 1: 选一首诗，列出它"写了什么"
Step 2: 列出它"没写什么"（至少找出5件应该写但没写的事）
Step 3: 分析每一处"没写"的功能——为什么这里需要空白？

示例（李商隐《夜雨寄北》）：

原文：
君问归期未有期，巴山夜雨涨秋池。
何当共剪西窗烛，却话巴山夜雨时。

【写了什么】：
1. 有人问我什么时候回去
2. 我不知道（未有期）
3. 现在在巴山，晚上下雨，秋天的池塘涨水了
4. 将来某天，我们会在西窗下一起剪烛芯
5. 那时候我会告诉你今天巴山夜雨的情形

【没写什么】：
1. "君"是谁？（妻子？朋友？恋人？→ 不确定 = 每个人都可以代入）
2. 我为什么在这里？（被贬？出差？避难？→ 不确定 = 开放解读）
3. 我现在的情绪是什么？（想家？孤独？无奈？安心？→ 全靠读者体会）
4. "何当"是什么时候？（明天？明年？永远不可能？→ 悬念）
5. 西窗在哪里？（家里？某个共同记忆的地方？→ 暗示"家"的存在）
6. 为什么是"剪烛芯"？（古代夜间照明的日常细节 → 温暖的家庭生活场景）
7. 最后一句的"巴山夜雨时"——为什么要重复？

【空白分析】：
- #1 的空白（"君"的身份不明）：让每个读者都能把自己的思念对象放进去
- #7 的重复（"巴山夜雨"出现两次）：第一次是"正在经历"，第二次是"将来回忆"——同一个短语在两句中的功能完全不同，这种重复创造了时间的回环感
- 整首诗的最大空白在"未来"：所有的希望都寄托在一个不确定的"何当"上——这个空白本身就是全诗的情感核心
```

### 训练3：古意新写练习（Classical Aesthetic + Modern Subject）

#### 目标

这是本技能最有价值的迁移能力：**用古典美学的方法处理现代题材**。

#### 方法论

```
什么是"古意新写"？

定义：使用古典诗词的美学原则（意境/留白/对仗/凝练），
      来书写当代的生活体验和情感。

不是"穿古装说现代话"（那是cosplay）。
是用古人看待世界的方式来看待今天的世界的。

核心转换公式：

[现代事物] → [古典审美转化] → [诗意表达]

转换策略：

策略A: 物品的"古典化命名"
  手机 → 掌中方寸 / 掌上萤 / 寸屏
  地铁 → 地龙 / 铁车 / 地下行舟
  电梯 → 升降梯 / 云梯 / 铁盒
  WiFi → 无形之网 / 空中脉络 / 电波
  外卖小哥 → 风中骑者 / 穿街人 / 绿衣使者

  ⚠️ 注意：不要生硬地替换！
  只有当替换后能增强诗意时才这样做。
  有时候直接用现代名词反而更有张力（古今碰撞的效果）。

策略B: 场景的"意境化处理"
  加班到深夜的办公室 → 
    "灯火三千窗，唯我一盏明"
    （化用"万家灯火"的意象，反转：别人都走了，只有我还亮着）
  
  在地铁上看手机 →
    "地底龙行十里暗，掌上光映一人颜"
    （地下=幽暗/隔绝，掌上光=手机屏幕=唯一的光源/连接外界的窗口）

策略C: 情感的"古典化表达"
  现代表达：我好焦虑啊，不知道未来怎么办
  古典转化：
    "前路茫茫何处问，心中鼓噪不成眠"
    （前路迷茫 + 内心焦躁如鼓鸣 + 失眠）

  现代表达：我想你了但我们不能在一起
  古典转化：
    "关山万重云阻隔，梦里依稀是旧容"
    （关山重重 = 距离/阻碍；梦里相见 = 唯一的相聚方式）
```

#### 练习任务

**题目**：用古典诗词的方式写以下5个现代场景

1. **场景**：凌晨三点的便利店
   **要求**：五言或七言绝句一首（20-28字）

2. **场景**：在高铁上看窗外飞逝的风景
   **要求**：一首小令（词牌自选，如《如梦令》《长相思》）

3. **场景**：看到手机里一张旧照片（已删除联系人的）
   **要求**：七言绝句一首

4. **场景**：周一早上的地铁拥挤
   **要求**：自由体裁，但要有古典韵味

5. **场景**：一个人吃火锅
   **要求**：自由体裁

**参考答案之一（场景1：凌晨三点的便利店）**：

```
方案A（七言绝句）：
长街灯火夜沉沉，独步寒门入店深。
货架千排皆过客，唯余温奶慰人心。
```

### 训练4：AI古典诗词质量门槛检测（Quality Threshold Detection）

#### 目标

建立一套标准来判断AI生成的古典诗词是否"合格"——注意，不是判断是否"优秀"，而是是否达到了"不像乱写的"这一最低门槛。

#### 质量门槛体系

```
Tier 0: 不合格（AI古典诗词的重灾区）
  表现：
  - 不守基本的格律（该押韵的不押韵、平仄混乱）
  - 用词严重时代错误（唐代诗人口中出现"愁云惨淡万里凝"这种明清小说语言）
  - 意象堆砌无逻辑（为了"看起来有诗意"而堆砌风花雪月）
  - 内容空洞无真情（全是套话，没有任何个人体验）
  
  例（典型的AI不合格输出）：
  "春风吹过百花香，蝴蝶飞舞在中央。
   我的心情很美丽，就像太阳一样亮。"
  问题诊断：
  - 完全不合律（这不是诗，是顺口溜）
  - "心情很美丽""像太阳一样亮" = 大白话，毫无古典美感
  - 没有任何独特的观察或感受

Tier 1: 合格（及格线——"像那么回事了"）
  表现：
  - 基本格律正确（押韵、平大致合规）
  - 用词在古典语汇范围内（没有明显的现代词汇污染）
  - 有一个或几个尚可的意象
  - 能传达一种大致的情绪（虽然可能比较泛）
  
  例（AI合格输出）：
  "独坐幽窗听夜雨，一灯如豆照孤身。
   故园千里梦难到，又是秋风落叶辰。"
  评价：
  - 格律基本合格（七言绝句，押韵正确）
  - 用词合格（幽窗、如豆、故园、辰等均为古典词汇）
  - 意象可用（夜雨、孤灯、故园、秋风、落叶）
  - 但：较为套路化（"听夜雨""照孤身""梦难到""落叶辰"都是常见组合）
  - 缺乏独特性和个人性

Tier 2: 良好（"可以拿出来给人看了"）
  表现：
  - 格律严谨无误
  - 有新鲜意象或新颖组合
  - 能感受到具体的个人情境（而非泛泛的"思乡""怀人"）
  - 至少有一处让人眼前一亮的表达
  
  例（AI良好输出）：
  "高楼三十层，我住云端第几重？
   推窗不见星，只见万家灯如涌。
   风从四面来，各带人间味——
   左耳炒锅香，右耳儿啼声。"
  评价：
  - 形式上有创新（不是标准的五七言，而是杂言）
  - "高楼三十层""云端第几重" = 现代事物的古典化表达
  - "万家灯如涌" = 新颖的比喻（灯像水一样涌动）
  - "左耳炒锅香，右耳儿啼声" = 极具体的生活细节，
    古典诗词中罕见这种都市烟火气的细腻捕捉
  - 有明确的个人在场感（我在高楼上、我能听到/闻到）

Tier 3: 优秀（"如果不说是AI写的，你可能以为是真人作品"）
  表现：
  - 格律完美（如果要写格律体的话）
  - 意境深远，有层次感
  - 语言精炼到每个字都不能换
  - 能引发多重解读
  - 有"属于这一个诗人"的独特声音
  
  （注：目前AI达到Tier 3的概率较低，但随着模型进步在逐步提高）
```

#### 快速检测清单

```
拿到一首AI生成的古典诗词后，按顺序检查：

□ 格律检查（如果是近体诗或词）
  - 押韵了吗？韵脚字是否属于同一韵部？
  - 平仄是否大体合规？（不必苛求完美，但不能明显违规）
  - 如果是词，是否符合所选词牌的字数和句式要求？

□ 语汇检查
  - 是否出现了明显的现代词汇？（除非是"古意新写"的刻意为之）
  - 用词是否准确？（有没有误用古典词汇的情况）
  - 是否有过于通俗的大白话混入？

□ 意象检查
  - 是否有至少一个"新鲜"的意象（非常见的搭配）？
  - 意象之间是否有逻辑关系（而非随机堆砌）？
  - 意象是否能唤起感官体验（而不只是抽象概念）？

□ 情感检查
  - 这首诗有"真感情"吗？（还是只是在"表演"感情？）
  - 你能感受到诗人当时的具体处境吗？
  - 还是只能感受到一个泛泛的"思乡/怀人/伤春悲秋"？

□ 整体评价
  - Tier 0: 重写
  - Tier 1: 可用但需修改
  - Tier 2: 很好
  - Tier 3: 优秀（保留并学习）
```

## 三、AI提示词公式（Classical Poetry Prompt Formula）

### 通用公式结构

```
[体裁指定] + [主题/触发点] + [风格约束] + [格律要求] + [禁忌清单]
= 完整的古典诗词生成提示词
```

### 公式详解

#### Component 1: 体裁指定（Genre Specification）— 写什么体裁

```markdown
=== 主要体裁选项 ===

【五言绝句】（5-character Quatrain）
  规格：4句，每句5字，共20字
  押韵：第2、4句必须押韵（第1句可押可不押）
  平仄：有严格规定（可要求AI遵守或放宽）
  特点：极其凝练，每字千金；适合捕捉瞬间感受
  代表作：王维《鹿柴》、柳宗元《江雪》

【七言绝句】（7-character Quatrain）
  规格：4句，每句7字，共28字
  押韵：同上
  特点：比五言稍从容，可容纳更多意象和信息
  代表作：杜牧《清明》、苏轼《题西林壁》

【五言律诗】（5-character Regulated Verse）
  规格：8句，每句5字，共40字
  押韵：偶数句押韵
  特殊要求：中间两联（第3-4句、第5-6句）必须对仗
  特点：结构完整，起承转合清晰；适合较复杂的主题
  代表作：杜甫《春望》、王湾《次北固山下》

【七言律诗】（7-character Regulated Verse）
  规格：8句，每句7字，共56字
  特点：近体诗中最长的标准体裁；容量最大
  代表作：杜甫《登高》、崔颢《黄鹤楼》

【词（Song Lyric）】
  规格：长短句，句数和字数因词牌而异
  常用词牌及特点：
  | 词牌 | 总字数 | 特点 | 适用情绪 |
  |------|--------|------|---------|
  | 如梦令 | 33字 | 短小轻灵 | 闲适、小确幸 |
  | 菩萨蛮 | 44字 | 上下阕结构 | 思念、婉约 |
  | 蝶恋花 | 60字 | 较长，可铺叙 | 离别、抒情 |
  | 水调歌头 | 95字 | 长调，容量大 | 哲理、豪放 |
  | 念奴娇 | 100字 | 最长常用词牌 | 怀古、壮阔 |
  | 鹧鸪天 | 55字 | 七言为主 | 叙事、感慨 |

【古体/古风】（Ancient Style Verse）
  规格：不受近体诗格律限制
  特点：句数不定（可长可短），字数不定（可杂言），
        不要求对仗，押韵较宽
  特点：最大自由度；适合叙事和抒发强烈情感
  代表作：《诗经》全部、《古诗十九首》、李白《蜀道难》
```

#### Component 2: 主题/触发点（Topic/Trigger）— 写什么内容

```markdown
原则：给"触发场景"而不是给"抽象主题"

❌ 差：
"写一首关于思乡的诗"
→ 太宽泛，AI会输出最通用的模板

✅ 好：

【触发A：具体场景型】
"触发场景：你在外地工作已经三年了。
今天下班路上看到一棵银杏树变黄了，
突然想起老家院子里也有这样一棵树，
小时候你经常在下面玩耍。
你想写一首关于'这个时刻'的诗。"

【触发B：物品触发型】
"核心意象：一把旧钥匙。
这把钥匙打不开任何门了（锁早就换了），
但你一直留着它。
因为它能打开你童年的家门。
请以'旧钥匙'为核心意象写一首五言绝句。"

【触发C：感官触发型】
"触发感官：雨后的泥土气味。
你在城市的公园里散步，
刚下过一场雨，
空气中有一股泥土的味道。
这个味道让你突然想起了小时候在乡下
一场大雨过后的那种感觉。
请以'雨后土香'为触发点写一首诗。"
```

#### Component 3: 风格约束（Style Constraints）

```markdown
=== 语言风格选项 ===

【风格A：盛唐气象】
  特征：开阔、雄浑、自信、自然
  关键词参考：雄浑、壮丽、飘逸、自然、天成
  代表诗人：李白、王维、孟浩然
  适用主题：山水、送别、边塞、饮酒

【风格B：沉郁顿挫】
  特征：深沉、厚重、忧愤、工整
  关键词参考：沉郁、顿挫、工整、凝重、悲悯
  代表诗人：杜甫
  适用主题：战乱、民生、怀古、自述

【风格C：婉约含蓄】
  特征：细腻、柔美、曲折、深情
  关键词参考：婉约、含蓄、柔美、曲折、深婉
  代表词人：李清照、柳永、晏殊
  适用主题：爱情、离愁、闺怨、惜春

【风格D：豪放旷达】
  特征：奔放、洒脱、哲理、气势
  关键词参考：豪放、旷达、潇洒、磅礴、超脱
  代表词人：苏轼、辛弃疾
  适用主题：咏史、抒怀、谈禅、壮志

【风格E：古意新写】（本技能特色）
  特征：古典的形式+现代的内容
  关键词参考：古今融合、以古写今、旧瓶新酒
  适用主题：都市生活、科技、现代情感
```

#### Component 4: 格律要求（Metric Requirements）

```markdown
根据选择的体裁，明确告知AI格律约束：

对于近体诗（绝句/律诗）：
  "严格遵守近体诗格律：
   - 平仄符合标准格式
   - 押韵使用平水韵（或中华新韵，需明确指定）
   - 律诗的颔联和颈联必须对仗"

对于词：
  "严格按照[词牌名]的格律填写：
   - 字数、句式、平仄、押韵位置均需符合词谱
   - 使用词林正韵（或中华新韵）"

对于古体/古风：
  "不需要遵守近体诗格律，但要保持古朴的语言风格
   和自然的节奏感。押韵可以较宽，但不要使用现代口语。"
```

#### Component 5: 禁忌清单（Prohibition List）

```markdown
🔴 绝对禁止：
  - 不要出现现代词汇（除非是"古意新写"的刻意安排）
    禁止词汇表：手机/电脑/互联网/地铁/咖啡/
                  打卡/emoji/yyds/绝绝子/躺平/内卷...
  - 不要使用大白话/口语化表达（如"我很想你""真好看"）
  - 不要使用成语（成语是"预制块"，诗要"现场制作"）
  - 不要在诗中出现解释性语句（"因为...所以..."）

🟡 尽量避免：
  - 不要使用过度滥用的意象组合：
    月亮+思念、秋天+悲伤、梅花+高洁、
    流水+时光、夕阳+迟暮...
  - 不要每句都以名词开头（造成单调感）
  - 不要为了凑字数而加入无意义的虚词
```

### 完整示例

#### 示例1：五言绝句·深夜加班（古意新写）

```
Prompt:
请写一首五言绝句，主题是"深夜加班"。

要求：
- 严格遵循五言绝句格律（押韵、平仄）
- 使用"古意新写"手法：用古典诗词的语言和美学
  来书写现代都市人的深夜加班体验
- 不要直接出现"加班""公司""电脑"等现代词汇
  （或者只用一次，作为古今碰撞的点睛之笔）
- 要有具体的意象（不是泛泛的"辛苦""疲惫"）
- 意境要准确传达出深夜办公室的氛围
- 20字，4句

参考风格：贾岛的苦吟 + 孟郊的寒士之风
```

**期望输出方向**：
```
案上灯如豆，窗间月似霜。
千行码未就，一夜鬓微苍。

解析：
- "案上灯如豆"：办公桌上的台灯（灯如豆=古典化的台灯描写）
- "窗间月似霜"：窗外月光（霜=寒冷/明亮/孤独的多重隐喻）
- "千行码未就"：代码写了上千行还没完成（"码"=code，
  唯一出现的轻微现代词汇，古今碰撞）
- "一夜鬓微苍": 一夜之间鬓角似乎有些苍白了（夸张的疲惫感）
- 格律：平仄基本合规，押"ang"韵（霜、苍）
```

#### 示例2：词·《如梦令》·外卖骑手

```
Prompt:
请填写一首《如梦令》，主题是"雨天骑电动车送外卖的骑手"。

《如梦令》词谱（33字）：
中仄中平平仄（韵），中仄中平平仄（韵）。
中仄仄平平，中仄平平仄（韵）。
中仄仄平，中仄仄平（韵）。
中仄，中仄（叠句），中仄中平平仄（韵）。

要求：
- 严格按词谱填写
- 用古典词的语言写现代骑手的体验
- 捕捉雨中骑行的具体感官细节（视觉/听觉/触觉）
- 情感基调：不是悲惨/卖惨，而是有一种坚韧的、
  默默坚持的生命力（类似宋代词人对普通生活的关注）
- 可以有一个"亮点"——一个让你印象深刻的细节
```

**期望输出方向**：
```
《如梦令·雨中骑》

细雨湿衣透骨，车轮碾水如沸。
红绿灯前停，单响催人心急。
穿行，穿行，温饭此时须抵。

解析：
- "细雨湿衣透骨"：雨渗透衣服冷到骨头（触觉+温度）
- "车轮碾水如沸"：车轮压过积水像煮沸一样（听觉+视觉+通感）
- "红绿灯前停"：唯一的"现代词汇"嵌入（红绿灯=古典词中的异质元素）
- "单响催人心急"：订单提醒声响，心里着急（"单"=订单的简称）
- "穿行，穿行"：叠句强化骑行的动态感
- "温饭此时须抵"：这时候那盒温热的饭是唯一的支撑
  （"须抵"= 必须依靠/支撑 = 骑手坚持下去的动力来源）
```

```
要求：
- 严格遵循七言律诗格律
```

## 四、评价标准（How to Judge Quality）

### 古典诗词五维自检清单

| 维度 | 1分（差） | 3分（及格） | 5分（优秀） | 自检方法 |
|------|----------|-----------|------------|---------|
| **格律准确性** | 明显不合律（错韵、平仄混乱、字数不对） | 大体合规但有少量瑕疵（可接受的微小偏差） | 完全合规，格律本身成为美感的一部分（如对仗工整得令人愉悦） | 对照词谱/诗律逐字检查；如有疑问可查《佩文韵府》或在线格律检查工具 |
| **语言纯正度** | 大量现代口语或时代错误的词汇混入 | 基本使用古典语汇，偶有不够精准的用词 | 用词精准、典雅，每个词都在其正确的历史语境中使用 | 标出所有"不太古典"的词；如果超过3个=扣分 |
| **意境深远度** | 只有表面意思，读完即忘，没有回味 | 有一定意境但较浅显，或意境较为套路化 | 有多层次的意义空间，每次读都有新的发现，能在心中停留很久 | 读完后闭上眼10秒——你还能"看见"或"感受到"什么吗？ |
| **意象质量** | 意象陈旧（风花雪月的老一套）或意象缺失（全是抽象概念） | 有具体意象但不够新鲜，或意象与情感的连接不够紧密 | 意象独特、精确、有感官冲击力；每个意象都是"这个诗人"独有的观察 | 盖住诗的一半，仅凭另一半的意象能否猜到大致主题？ |
| **情感真实性** | 情感虚假或表演性的（"为赋新词强说愁"） | 有真情但表达略显刻意或不够深入 | 情感像是从生命经验中直接提取的——不加工不美化但因为真实所以动人 | 读的时候你的身体有反应吗？（皮肤/呼吸/心跳）——终极检测标准 |

### 快速评分流程

```
生成诗词后，按以下步骤快速自检（3分钟）：

Step 1: 格律速查（30秒）
  → 押韵了吗？韵脚字是否属于同一韵部？
  → 如果是词/律诗，对仗了吗？
  → 平仄是否有明显违和之处？
  → 格律维度评分

Step 2: 语感朗读（30秒）
  → 出声朗读（或默读但有节奏感）
  → 朗朗上口吗？还是有"卡"的地方？
  → "卡"的地方通常是格律或用词有问题
  → 语言维度评分

Step 3: 意象清点（30秒）
  → 数一数有多少个独立的、具体的意象
  → < 3个 = 意象维度 ≤ 2分
  → 3-5个 = 3-4分
  → > 5个且每个都有足够的空间 = 5分

Step 4: "翻译测试"（60秒）
  → 尝试把它"翻译"成现代白话散文
  → 如果翻译后发现"原来就这点意思"= 意境不足
  → 如果翻译后发现"翻不出来/翻译后丢失了大量信息"=
    原诗的密度很高（加分项）

Step 5: 综合判断
  → 平均分 ≥ 4.0：优秀
  → 3.0-3.9：良好，针对性修改
  → < 3.0：需要重新构思
```

## 五、常见陷阱与修正（Pitfalls & Fixes）

| # | 陷阱 | 典型表现 | 原因 | 修正方案 |
|---|------|---------|------|---------|
| P1 | **格律崩坏** | 该押韵的不押韵、平仄完全无视、字数不对 | AI对古典格律的理解不稳定；缺少明确的格律指令 | 在提示词中明确写出词谱或诗律；提供格律正确的范例作为few-shot；使用在线格律检查工具辅助验证 |
| P2 | **时代错乱** | 唐代诗人口中说"愁云惨淡万里凝"；宋代词人用网络流行语 | AI的训练数据混合了不同时代的文本；缺乏"时代意识" | 明确指定朝代/时期风格；提供"禁用词汇表"；强调"语言风格的纯一度" |
| P3 | **意象堆砌** | 为了"有诗意"而大量堆砌风花雪月、梅兰竹菊 | AI认为"诗意"="古典意象的数量" | 限制意象数量（全诗≤5个独立意象）；要求每个意象必须有独特的功能 |
| P4 | **无情而矫情** | 没有真实的情感基础，强行"演出"悲伤或喜悦 | AI没有真实人生体验；倾向于模仿"情感表达的模板" | 注入具体的人格设定和经历背景；要求"基于真实体验"而非"基于诗歌套路" |
| P5 | **说理破坏意境** | 在写景/抒情的过程中突然插入议论或说教 | AI受宋代理学诗影响（"文以载道"的传统） | 明确禁止"议论""说理""讲道理"；要求"一切通过意象来表达" |
| P6 | **用典不当** | 错误使用或不恰当地堆砌典故 | AI对典故的理解停留在表面；为了"显得有学问"而滥用 | 限制典故数量（全诗≤1个）；如果用典必须确保准确；初学者建议完全不用典 |
| P7 | **失去"古味"** | 形式上是古诗，但读起来像穿了古装的白话文 | AI的底层语言模型是现代汉语；古典语感的培养需要专门训练 | 提供"古文范本"作为风格参考；增加"古文虚词"的使用要求（之乎者也/兮/矣/焉/哉）；要求"避免使用现代语法结构" |

### 陷阱深度解析：P1 格律崩坏

```
问题诊断：

  AI最常见的格律错误类型：

  错误类型A: 押韵错误
    ❌ "春风吹过花自开（kai），我心欢喜乐悠哉（zai），
        明日再去踏青来（lai）。"
    分析：开/哉/lai 在平水韵中属于不同韵部（"开"属灰韵，
          "哉""lai"属皆韵），虽然现代普通话读音相近，
          但在古典诗词中不算押韵。
    ✅ 修正：统一韵部，如改为"花自开（kai）- 乐怀（huai）-
        踏青来（lai）"（皆韵内部通押）

  错误类型B: 平仄错误
    ❌ 七言绝句第一句应该是"（仄）仄（平）平仄仄平"或
        "（平）平（仄）仄仄平平"
        AI输出："今天天气真正好（平平平仄平仄仄）"——
        完全不合律。
    ✅ 修正：在提示词中给出具体的平仄格式；
        或使用"宽松格律"模式（只要求押韵和节奏感，
        不强求严格的平仄）

  错误类型C: 字数错误
    ❌ 说好写五言绝句，结果有一句变成了6个字或4个字
    ✅ 修正：在提示词中明确标注"每句 exactly 5个汉字"；
        生成后逐句数字数

  错误类型D: 词牌格式错误
    ❌ 填写《如梦令》但句式完全不对
        （如把33字的《如梦令》写成了50字）
    ✅ 修正：在提示词中完整写出词谱（包括每句字数和韵脚标记）；
        这是词体创作中最不可妥协的硬性约束
```

## 六、跨域迁移（Cross-Domain Transfer）

> **品味的最高级应用：把在一个领域学到的东西用到另一个完全不同的领域。**

### 迁移地图

| 目标领域 | 迁移什么？ | 怎么迁移？ | 示例 |
|---------|-----------|-----------|------|
| **现代歌词** | 古典诗词的凝练和对仗技巧 | 歌词本质上是有旋律的诗；古典的炼字功夫可以直接提升歌词品质 | 方文山的歌词（周杰伦御用作词人）深受古典诗词影响；《青花瓷》《兰亭序》等直接以古典为主题 |
| **品牌命名/文案** | 古典的雅致和内涵 | 好的品牌名字像一句好的诗词——简短、有意味、耐寻味 | "百度"出自辛弃疾"众里寻他千百度"；"华为"无直接出处但有古典气质 |
| **产品设计** | "少即是多"的古典哲学 | 古诗的极致凝练 = 设计的极简主义；每一个元素都必须有存在的理由 | Apple的产品命名和设计理念与古典美学有深层共鸣 |
| **建筑/园林** | 意境营造和借景手法 | 中国古典园林的设计哲学直接来源于诗词画的"意境"理论 | 苏州园林的"移步换景""借景""框景" = 立体的诗词 |
| **游戏设计** | 世界观构建的古典方法 | 用古典诗词的方式构建游戏的叙事氛围和文化底蕴 | 《仙剑奇侠传》系列、《古剑奇谭》、《原神》的璃月地区深受古典美学影响 |

### 具体迁移练习：古典式品牌文案

```
任务：为一个新兴的中国茶饮品牌写一段品牌宣言

普通版（现代广告文案）：
"XX茶饮，精选高山茶园的优质茶叶，
采用传统工艺精心制作，每一杯都蕴含着
大自然的精华和中国千年茶文化的底蕴。
忙碌生活中的一杯XX茶，让您品味东方之美。"

古典意境版（迁移留白+凝练+意象）：
"山不在高，有茶则灵。
水不在深，有味则馨。
一杯之中，有云雾，有晨曦，有采茶人的指尖温度。
不必言语。
饮者自知。"

分析：
- 原版：80+字，信息密度低，全是抽象赞美
- 古典版：36字，信息密度极高
- 引用刘禹锡《陋室铭》的开篇结构（"山不在高..."）
  → 借力经典，瞬间获得文化厚度
- "云雾""晨曦""指尖温度" = 三个具体意象
  → 替代了所有形容词
- "不必言语。饮者自知。" = 四个字的留白
  → 信任消费者自己的感受能力
  → 比"让您品味东方之美"有力一百倍
```
