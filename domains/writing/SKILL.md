## 三级技能索引

### 📜 诗歌 Poetry (`poetry/`)

**2个诗歌方向，从现代自由诗到古典意境重构。**

| 技能路径 | 方向 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|------|-----------|------------|-----------|
| `poetry/modern-chinese` | 现代汉语诗歌 | 从日常中提炼诗意 | 现代诗创作/情感表达/意象实验；追求自由形式和当代语言节奏 | "现代诗""新诗""自由诗""意象""当代诗歌" |
| `poetry/classical-chinese` | 古典诗词意境重构 | 古意新写的审美转换 | 古风写作/古诗词新编/传统文化表达；需要典雅韵味但用现代语言可读 | "古诗词""古风""意境""格律""唐宋""中国风" |

#### 诗歌技能选择速查表

```
我想写...                        → 推荐技能
─────────────────────────────────────────────
一首关于当下生活的现代诗           → modern-chinese
有古典韵味的诗意表达（非严格格律）→ classical-chinese
```

**诗歌 vs 其他文体的边界**

```
纯抒情/意象密集/短小精悍        → poetry/
有叙事线/人物/场景              → prose/ 或 screenplay/
说服性/商业性                   → copywriting/
```

### 📖 散文 Prose (`prose/`)

**3位散文大师，覆盖极简、魔幻、都市三大叙事风格。**

| 技能路径 | 大师风格 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|---------|-----------|------------|-----------|
| `prose/hemingway-minimal` | 欧内斯特·海明威 · 极简白话 | 冰山理论的文字实践 | 动作描写/硬汉叙事/简洁有力的文体；需要克制、留白、让读者自己补全意义 | "海明威""极简""短句""硬汉""冰山理论""克制" |
| `prose/magical-marquez` | 加西亚·马尔克斯 · 魔幻现实主义 | 神奇与现实的无缝编织 | 拉美风情/家族史诗/超现实日常；需要将不可思议的事物写得像真的一样自然 | "马尔克斯""魔幻现实""孤独""百年孤独""拉美" |
| `prose/urban-murakami` | 村上春树 · 都市孤独美学 | 小确丧的精确表达 | 都市生活/孤独感/爵士氛围/隐喻系统；需要捕捉现代人微妙的疏离感和内心独白 | "村上春树""都市""孤独""爵士""隐喻""小确幸/小确丧" |

#### 散文技能选择速查表

```
我想写...                        → 推荐技能
─────────────────────────────────────────────
简洁有力、动作驱动的叙事           → hemingway-minimal
充满想象力、神奇与现实的交织       → magical-marquez
都市人的内心独白、微妙情绪         → urban-murakami
```

**三位大师的风格对比**

| 维度 | 海明威 | 马尔克斯 | 村上春树 |
|------|-------|---------|---------|
| **句式** | 短句为主，主谓宾清晰 | 长句，层叠的从句 | 中等长度，大量精确细节 |
| **情感表达** | 克制，冰山下隐藏 | 浓烈，直接呈现奇迹 | 微妙，通过物品和音乐暗示 |
| **典型主题** | 战争、死亡、勇气 | 家族、命运、时间 | 孤失、寻找、日常仪式 |
| **适合篇幅** | 短篇至中篇最佳 | 长篇史诗 | 中篇至长篇 |
| **AI生成难度** | ⭐⭐（简单直接） | ⭐⭐⭐⭐（需维持魔幻平衡） | ⭐⭐⭐（需构建隐喻系统） |

### 🎬 剧本 Screenplay (`screenplay/`)

**2种剧本风格，从复杂结构到情绪碎片。**

| 技能路径 | 风格类型 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|---------|-----------|------------|-----------|
| `screenplay/cinema-nolan` | 克里斯托弗·诺兰式复杂叙事 | 时间结构与观众智力博弈 | 悬疑片/科幻/非线性叙事；需要精密结构和多重解读空间 | "诺兰""剧本""复杂叙事""非线性""悬疑""时间结构" |
| `screenplay/wong-mood-fragment` | 王家卫式情绪碎片 | 独白即剧情，氛围即故事 | 文艺片/爱情片/都市情感；需要强烈的个人风格和情绪氛围 | "王家卫""独白""情绪""碎片化""文艺片""港风" |

#### 剧本技能选择速查表

```
我想写...                        → 推荐技能
─────────────────────────────────────────────
一个有时间诡计或悬疑结构的剧本     → cinema-nolan
一个充满情绪独白和氛围感的文艺剧本 → wong-mood-fragment
```

**剧本 vs 散文的关键区别**

```
Screenplay = 视觉化 + 对话驱动 + 场景切换
Prose = 内心独白 + 描写自由 + 时间线性（通常）

如果你的故事：
- 更依赖画面和对白推进    → screenplay/
- 需要深入人物内心世界     → prose/
```

### 💼 文案 Copywriting (`copywriting/`)

**2种文案流派，从经典广告到社交时代。**

| 技能路径 | 流派 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|------|-----------|------------|-----------|
| `copywriting/ogilvy-classic` | 大卫·奥格威经典广告学派 | 用事实和洞察打动人心 | 品牌长文案/产品详情页/说服性内容；需要逻辑严密、事实支撑、建立信任 | "广告文案""奥格威""经典""长文案""说服""品牌" |
| `copywriting/digital-native` | 数字原生文案 | 社交时代的注意力捕获 | 社交媒体/短视频脚本/标题党/互动文案；需要在3秒内抓住注意力并引发行动 | "社交媒体""短视频文案""标题党""互动""病毒传播""数字化" |

#### 文案技能选择速查表

```
我想写...                        → 推荐技能
─────────────────────────────────────────────
一篇有深度的品牌/产品说服文案      → ogilvy-classic
一条能在社交平台快速传播的内容     → digital-native
```

**两种文案的核心差异**

| 维度 | 奥格威经典 | 数字原生 |
|------|----------|---------|
| **目标** | 说服理性决策 | 触发即时行动 |
| **篇幅** | 长篇（500-2000字） | 超短（10-140字） |
| **核心武器** | 事实 + 洞察 + 信任 | 钩子 + 情绪 + CTA |
| **典型渠道** | 杂志/官网/邮件 | 微信/抖音/Twitter |
| **成功指标** | 转化率/品牌认知 | 点击率/分享率/互动量 |
| **适用场景** | B2B/高客单价产品 | B2C/快消品/UGC |

## 选择决策树："我想写什么类型？"

当用户说"帮我写XX"但不确定具体风格时，使用此决策树：

```
开始：用户的写作需求是什么？
│
├─ 目的类型判断
│  ├─ "抒发情感/表达感受"        → 进入 [抒情分支]
│  ├─ "讲述故事/塑造人物"        → 进入 [叙事分支]
│  ├─ "视觉化呈现/影视剧本"      → 进入 [剧本分支]
│  ├─ "销售/推广/传播"          → 进入 [商业分支]
│  └─ "不确定/混合需求"          → 进入 [通用诊断]
│
├─ [抒情分支] （诗歌）
│  ├─ 想用现代语言？             → modern-chinese (现代汉语诗)
│  └─ 想有古典韵味？            → classical-chinese (古典诗词)
│
├─ [叙事分支] （散文）
│  ├─ 想要简洁有力？             → hemingway-minimal (海明威)
│  ├─ 想要奇幻色彩？             → magical-marquez (马尔克斯)
│  └─ 想要都市感/现代感？        → urban-murakami (村上春树)
│
├─ [剧本分支]
│  ├─ 需要复杂结构/悬疑？         → cinema-nolan (诺兰)
│  └─ 需要情绪氛围/文艺感？       → wong-mood-fragment (王家卫)
│
├─ [商业分支] （文案）
│  ├─ 长文案/深度说服？           → ogilvy-classic (奥格威)
│  └─ 短文案/社交传播？           → digital-native (数字原生)
│
└─ [通用诊断] （5个快速问题）
   Q1: 给谁看？（自己→诗歌/散文 | 大众→文案 | 专业圈→散文/剧本）
   Q2: 多长？
       → <100字 → 诗歌/数字文案
       → 100-2000字 → 散文/奥格威文案
       → >2000字 → 长篇散文/完整剧本
   Q3: 情绪目标？
       → 温暖治愈 → 村上春树/吉卜力（跨域）
       → 震撼有力 → 海明威/诺兰
       → 神秘梦幻 → 马尔克斯/达利（跨域）
       → 幽默轻松 → 数字原生文案
   Q4: 形式偏好？
       → 自由流动 → 现代诗/马尔克斯
       → 结构严谨 → 海明威/诺兰/奥格威
       → 碎片化/意识流 → 王家卫/村上春树
   Q5: 商业还是纯艺术？
       → 商业目的 → copywriting/
       → 纯艺术表达 → poetry/prose/screenplay/
```

## 与 AI 写作大师的协作流程

> **本领域的所有技能都需要配合 `ai-writing-master` 才能完成从意图到文本的完整闭环。**

### 协作架构

```
用户需求："用海明威的笔法写一段关于加班的感受"
│
├─ Step 1: 加载文学大师技能 (品味层)
│  └── prose/hemingway-minimal/SKILL.md
│      ├── 提供：冰山理论哲学
│      ├── 提供：感知训练（如何识别"可省略的部分"）
│      ├── 提供：海明威式句法模板
│      └── 提供：极简主义评价标准
│
├─ Step 2: 构建"品味+工艺"提示词
│  └── 结合：
│      ├── 大师的文体特征（来自Step 1）
│      └── AI写作平台的指令工程（见下方）
│
├─ Step 3: AI写作大师执行 (工艺层)
│  └── ai-writing-master/SKILL.md
│      ├── 选择平台（ChatGPT? Claude? Gemini?）
│      ├── 将文体特征转化为system prompt
│      ├── 生成初稿
│      └── 自动迭代润色（最多5轮）
│
└─ Step 4: 文学大师评价 (回到品味层)
   └── 使用海明威的自检清单评分
       ├── 句子平均长度 ✓ (<15字)
       ├── 主动语态比例 ✓ (>80%)
       ├── 形容词/副词密度 ✓ (<5%)
       ├── 冰山效果 ✓（未言之意可感知）
       └── 节奏感 ✓（长短句交替）
       → 输出最终文本 + 评价报告
```

### 不同平台的适配建议

| 目标平台 | 推荐搭配技能 | 备注 |
|---------|-------------|------|
| **ChatGPT-4o** | 所有技能（通用性强） | 最擅长模仿风格，适合创意写作 |
| **Claude 3.5** | 长文本/散文/剧本 | 擅长长程一致性，适合小说和剧本 |
| **Gemini Pro** | 研究型写作/非虚构 | 擅长整合信息，适合奥格威式事实型文案 |
| **中文专用模型** (文心一言/通义千问) | 古典诗词/中文散文 | 中文语境更地道，适合中文母语风格 |

## 典型工作流示例

### 示例1：新手首次使用 — 写一首现代诗

```
用户："我想写一首关于城市的孤独感的诗"

系统响应：
  1. 路由判断：[抒情] → [现代语言] → modern-chinese
  2. 加载 poetry/modern-chinese/SKILL.md
  3. 引导完成：
     - 感知热身（5分钟）：读北岛/顾城的代表作，练习"意象提炼"
     - 提示词构建（使用现代诗公式）：
       "[主体: 城市] + [核心意象: 霓虹灯/地铁/窗户] +
        [情绪: 孤独但不绝望] + [手法: 通感/隐喻]"
     - 调用 ai-writing-master 生成3版
     - 用现代诗自检清单选最佳版本：
       ✓ 意象是否新鲜（避免陈词滥调）
       ✓ 节奏是否有音乐性
       ✓ 是否留有想象空间（不说破）
```

### 示例2：熟练用户 — 商业文案

```
用户："帮一款智能手表写奥格威风格的电商详情页文案"

自动路由：
  → copywriting/ogilvy-classic/SKILL.md (品味层)
  → ai-writing-master/SKILL.md (工艺层)

执行流程：
  1. 加载奥格威的"长文案公式"：
     - 标题：承诺具体利益（不是花哨的）
     - 开头：用事实/数据建立权威
     - 正文：逐层展开卖点，每个都有证据
     - 结尾：明确的行动号召
  2. AI大师执行：生成初稿（约1500字）
  3. 奥格威自检：
     ✓ 标题是否承诺了具体利益？（✓ "每天为你节省47分钟的智能助手"）
     ✓ 是否避免了空洞形容词？（✓ 删除了"革命性""颠覆性"等）
     ✓ 是否有可信的证据？（✓ 添加了用户数据和专家背书）
     ✓ 行动号召是否明确？（✓ "立即体验，30天免费试用"）
  4. 输出最终文案 + A/B测试建议
```

### 示例3：跨域融合（高级）

```
用户："写一段王家卫式的独白，但是讲一个科幻故事"

融合模式：wong-mood-fragment(主) + cinema-nolan(辅)
  主技能：王家卫的情绪碎片法（70% - 独白口吻/时间标记/物象隐喻）
  辅技能：诺兰的时间结构思维（30% - 非线性/记忆不可靠/科学概念）

输出示例（片段）：
  "2067年6月4日下午3点，距离那次量子纠缠事故已经过去
   了4年零17天。我依然会在每个雨天去买同一罐凤梨罐头，
   只是因为它的保质期是2071年5月1日。

   物理学家说时间是第四维度，但我总觉得它更像是一卷
   被反复播放又磨损的老胶片。每一次回溯，画面的颗粒
   都会更粗一些，她的声音也更模糊一分。

   也许在某个平行宇宙里，我没有按下那个按钮。
   但在这个宇宙，我只能一遍遍地吃这罐过期的甜，
   等待一个永远不会来的电话。"

评价：
  ✓ 王家卫元素：时间标记（日期/保质期）/ 物象（罐头）/ 独白口吻
  ✓ 诺兰元素：量子纠缠/平行宇宙/时间作为维度的科学隐喻
  ✓ 融合度：自然，没有违和感
```

## 领域特色功能

### 1. 感知训练库

每个技能都包含独立的感知训练模块：

| 训练类型 | 对应技能 | 训练内容 | 时长 |
|---------|---------|---------|------|
| 意象提炼训练 | 现代汉语诗 | 从日常事物中发现诗意 | 10分钟 |
| 格律敏感度训练 | 古典诗词 | 识别平仄和韵脚的音乐性 | 15分钟 |
| 冰山识别训练 | 海明威 | 练习找出"可以删掉而不影响理解的句子" | 15分钟 |
| 魔幻平衡训练 | 马尔克斯 | 将荒诞事件写得像新闻一样平淡 | 20分钟 |
| 隐喻系统构建 | 村上春树 | 建立个人化的象征符号库 | 25分钟 |
| 结构解构训练 | 诺兰 | 分析电影的时间线和伏笔网络 | 30分钟 |
| 独白节奏训练 | 王家卫 | 练习口语化但有诗意的独白 | 15分钟 |
| 洞察挖掘训练 | 奥格威 | 从产品中找到真正独特的卖点 | 20分钟 |
| 注意力钩子训练 | 数字原生 | 在3秒内制造好奇心缺口 | 10分钟 |

### 2. 风格迁移矩阵

已验证的跨技能组合：

| 主技能 × 辅技能 | 效果描述 | 难度 | 适用场景 |
|---------------|---------|------|---------|
| 海明威句式 × 马尔克斯题材 | 用极简文字写魔幻故事 | ⭐⭐⭐ | 实验小说 |
| 村上春树隐喻 × 王家卫独白 | 双重孤独叠加 | ⭐⭐ | 都市文艺作品 |

## 写作伦理提醒

在使用AI辅助写作时，请注意：

1. **原创性声明**：如使用AI辅助，建议适当标注（根据发布平台要求）
2. **避免抄袭**：即使AI生成的，也要确保不无意中复制现有作品
3. **事实核查**：AI可能产生幻觉，尤其是非虚构类写作务必核实
4. **声音保持**：AI是工具，最终应该听起来像**你**，而不是像大师的拙劣模仿
5. **版权意识**：了解目标平台对AI生成内容的版权政策

---

> **记住：这个领域不是教你"怎么凑字数"，而是教你怎么"像大师一样思考和感知"。**
> **技巧可以学习，但品味只能通过大量的阅读和练习来培养。**
> **AI帮你写得更快，但只有你能决定写什么值得被写下来。**
