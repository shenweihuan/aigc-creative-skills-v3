## 三级技能索引

### 🖼️ 绘画 Painting (`painting/`)

**6个大师级绘画风格，覆盖西方美术史核心流派。**

| 技能路径 | 大师/流派 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|----------|-----------|------------|-----------|
| `painting/impressionist-monet` | 克劳德·莫奈 · 印象派 | 光的流动捕捉者 | 风景/静物/需要温暖氛围感的场景；色彩丰富但不过于艳俗的作品 | "印象派""光影""莫奈""色彩丰富""温柔" |
| `painting/renaissance-da-vinci` | 列奥纳多·达芬奇 · 文艺复兴 | 完美比例与神圣构图 | 人物肖像/宗教主题/需要古典庄严感；追求精确解剖和黄金分割 | "古典""文艺复兴""达芬奇""人体""完美比例" |
| `painting/modernist-picasso` | 巴勃罗·毕加索 · 立体主义 | 打碎现实再重组 | 抽象表达/现代艺术/多视角叙事；想要打破常规、制造视觉冲击 | "立体主义""几何""抽象""毕加索""多视角" |
| `painting/surrealist-dali` | 萨尔瓦多·达利 · 超现实主义 | 梦境逻辑具象化 | 超现实场景/梦境描述/荒诞隐喻；需要制造神秘感和想象空间 | "超现实""梦境""达利""荒诞""融化时钟" |
| `painting/landscape-turner` | 约瑟夫·马洛德·透纳 · 浪漫主义风景 | 大气与光的史诗 | 大幅风景/海景/戏剧性天空；追求壮阔感和光影戏剧性 | "风景画""透纳""大气透视""海景""壮阔" |
| `painting/portrait-rembrandt` | 伦勃朗·范·莱因 · 巴洛克肖像 | 明暗对照的灵魂刻画 | 人物肖像/心理深度/戏剧性光感；需要表现人物内在情感 | "肖像""伦勃朗""光感""明暗对照""油画质感" |

#### 绘画技能选择速查表

```
我想画...                        → 推荐技能
─────────────────────────────────────────────
一张温暖的风景（日出/日落）        → impressionist-monet

抽象/几何/打破常规的现代作品       → modernist-picasso
梦境/超现实的奇幻场景            → surrealist-dali
壮阔的大自然（大海/山脉）         → landscape-turner
有灵魂深度的肖像                 → portrait-rembrandt
```

## 选择决策树："我想画什么风格？"

当用户说"我想画一张XX"但不确定具体风格时，使用此决策树：

```
开始：用户想创作什么类型的视觉作品？
│
├─ 类型判断
│  ├─ "风景/自然/户外"           → 进入 [风景分支]
│  ├─ "人物/肖像/人脸"           → 进入 [人物分支]
│  ├─ "抽象/概念/想象力"         → 进入 [抽象分支]
│  └─ "其他/不确定"              → 进入 [通用诊断]
│
├─ [风景分支]
│  ├─ 想要温暖/柔和的感觉？      → impressionist-monet (莫奈)
│  ├─ 想要壮阔/戏剧性的感觉？    → landscape-turner (透纳)
│  ├─ 想要极致技术/黑白？        → bw-landscape-adams (亚当斯)
│  └─ 想要宁静/时间感？          → minimalist-sugimoto (杉本博司)
│
├─ [人物分支]
│  ├─ 古典/完美比例？            → renaissance-da-vinci (达芬奇)
│  ├─ 心理深度/戏剧光感？        → portrait-rembrandt (伦勃朗)
│  ├─ 现代/创意/戏剧性？         → portrait-leibovitz (莱博维茨)
│  └─ 动漫/可爱风格？            → anime-miyazaki (吉卜力)
│
├─ [抽象分支]
│  ├─ 几何/解构/现代感？         → modernist-picasso (毕加索)
│  ├─ 梦境/超现实/荒诞？         → surrealist-dali (达利)
│  └─ 奇幻/史诗/游戏感？         → concept-art-frazetta (概念设计)
│
└─ [通用诊断] （4个快速问题）
   Q1: 受众是谁？（大众→莫奈/吉卜力 | 专业→毕加索/亚当斯）
   Q2: 情绪目标？（温暖→莫奈 | 震撼→透纳/毕加索 | 神秘→达利 | 宁静→杉本博司）
   Q3: 精确度偏好？
       → 写实精确 → 达芬奇/亚当斯/伦勃朗
       → 有控制的暗示 → 莫奈/透纳
       → 打破规则 → 毕加索/达利
   Q4: 时间约束？
       → 快速产出 → 莫奈/吉卜力（关键词相对简单）
       → 愿意打磨 → 任何大师都适合
```

## 与 AI 绘画大师的协作流程

> **本领域的所有技能都需要配合 `ai-painting-master` 才能完成从意图到图像的完整闭环。**

### 协作架构

```
用户需求："用莫奈的风格画一张秋天的湖边小屋"
│
├─ Step 1: 加载艺术大师技能 (品味层)
│  └── impressionist-monet/SKILL.md
│      ├── 提供：莫奈的色彩哲学
│      ├── 提供：感知训练（如何看光）
│      ├── 提供：莫奈专属关键词库
│      └── 提供：印象派评价标准
│
├─ Step 2: 构建"品味+工艺"提示词
│  └── 结合：
│      ├── 艺术大师的风格定义（来自Step 1）
│      └── AI绘画大师的平台适配（见下方）
│
├─ Step 3: AI绘画大师执行 (工艺层)
│  └── ai-painting-master/SKILL.md
│      ├── 选择平台（Midjourney? DALL-E? Stable Diffusion?）
│      ├── 将品味转化为具体参数
│      ├── 生成初稿
│      └── 自动迭代优化（最多5轮）
│
└─ Step 4: 艺术大师评价 (回到品味层)
   └── 使用印象派的自检清单评分
       ├── 色彩和谐度 ✓
       ├── 光线捕捉 ✓
       ├── 笔触感 ✓
       └── 整体氛围 ✓
       → 输出最终图像 + 评价报告
```

### 不同平台的适配建议

| 目标平台 | 推荐搭配技能 | 备注 |
|---------|-------------|------|
| **Midjourney v6** | 所有绘画/插画技能 | 最适合 artistic style，支持 --stylize 参数 |
| **DALL-E 3** | 达芬奇/伦勃朗/莱博维茨 | 擅长写实和精确描述，适合古典和肖像 |
| **Stable Diffusion** | 所有技能（需配LoRA） | 可控性最强，适合专业用户，推荐配合 ControlNet |
| **Adobe Firefly** | 商业插画/概念设计 | 安全版权，适合商业项目 |

## 领域特色功能

### 1. 感知训练库

每个技能都包含独立的感知训练模块，不是教你技术，而是训练你的眼睛：

| 训练类型 | 对应技能 | 训练内容 | 时长 |
|---------|---------|---------|------|
| 色彩分解训练 | 莫奈 | 学会将复杂色彩拆解为纯色斑块 | 10分钟 |
| 比例与构图训练 | 达芬奇 | 黄金分割螺旋的实际应用 | 15分钟 |
| 多视角观察训练 | 毕加索 | 同时从3个角度观察同一物体 | 10分钟 |
| 梦境记录训练 | 达利 | 捕捉潜意识意象的方法 | 15分钟 |
| 光影层次训练 | 透纳/伦勃朗 | 识别5级明暗变化 | 10分钟 |
| 决定性瞬间预判 | 布列松 | 预测最佳拍摄时机 | 20分钟 |

## 本领域边界与扩展方向

### 当前覆盖范围 ✅
- 西方美术史核心流派（文艺复兴→印象派→立体主义→超现实主义）
- 摄影史四大支柱（风光/纪实/人像/当代艺术摄影）
- 主流商业插画（日系动画/概念设计）

### 计划扩展 📋
- **东方绘画**：中国水墨（齐白石/张大千）、日本浮世绘（葛饰北斋）
- **当代艺术**：波普艺术（安迪·沃霍尔）、极简主义（马克·罗斯科）
- **数字艺术**：赛博朋克、蒸汽波、像素艺术
- **3D渲染**：Blender/C4D风格的AI生成
- **更多摄影**：时尚摄影（赫尔穆特·牛顿）、新闻摄影（罗伯特·卡帕）

## 快速开始指南

### 你是第一次来？

```
推荐入门路径（约45分钟完成全流程）：

1. 选择一个你感兴趣的方向（绘画/摄影/插画）
2. 从该方向选一个最吸引你的大师
3. 阅读 SKILL.md 的"哲学奠基"部分（5分钟）
4. 完成"感知热身"训练（10-15分钟）
5. 用提供的"提示词公式"生成第一张图（5分钟）
6. 用"自检清单"评价并迭代（10-15分钟）
7. 记录心得，准备下一次创作

推荐新手首选技能：
- 想轻松上手 → impressionist-monet（莫奈，容错率高，效果好）
- 想学技术 → renaissance-da-vinci（达芬奇，基础扎实）
- 想玩创意 → surrealist-dali（达利，有趣且震撼）
```

### 你已经熟悉这个领域？

```
进阶玩法：

1. 尝试跨技能融合（见上方"风格迁移矩阵"）
2. 深入一位大师：完成全部感知训练 + 创作5件作品
3. 开始跨域迁移实验：
   - 用莫奈的色彩逻辑去写一首诗（→ writing/poetry/）
   - 用亚当斯的构图思维去做UI设计（→ design/graphic/）
4. 探索计划中的新技能（或自己造一个新的！）
```
