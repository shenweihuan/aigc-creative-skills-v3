## 三级技能索引

### 🎭 导演 Directing (`directing/`)

**3位电影大师，覆盖史诗、后现代、诗电影三大导演美学。**

| 技能路径 | 大师 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|------|-----------|------------|-----------|
| `directing/epic-kubrick` | 斯坦利·库布里克 | 每一帧都是精心计算的构图 | 史诗题材/科幻片/需要极致对称和完美主义；追求冷峻、精密、有仪式感的视觉语言 | "库布里克""史诗""对称""冷峻""完美主义""2001太空漫游" |
| `directing/non-linear-tarantino` | 昆汀·塔伦蒂诺 | 非线性叙事的暴力美学 | 后现代叙事/对话驱动/类型片解构；想要打破时间线、大量对白、黑色幽默 | "昆汀""非线性""对话驱动""暴力美学""反转""低俗小说" |
| `directing/tarkovsky-poetic` | 安德烈·塔可夫斯基 | 把时间雕刻成诗 | 艺术电影/长镜头/灵性主题；追求慢节奏、诗意、形而上思考、时间雕塑 | "塔可夫斯基""诗电影""长镜头""灵性""慢节奏""镜子" |

#### 导演技能对比矩阵

| 维度 | 库布里克 | 昆汀 | 塔可夫斯基 |
|------|---------|------|----------|
| **核心美学** | 对称/单点透视/仪式感 | 类型混搭/非线性/元电影 | 长镜头/诗意蒙太奇/记忆与梦 |
| **时间处理** | 线性但极度精确 | 打碎重组（倒叙/插叙/多线） | 时间作为实体（延展/凝固） |
| **镜头语言** | 广角/推轨/极端对称 | 中景/特写/静态对话 | 深焦/横移/极长镜头（数分钟） |
| **声音设计** | 古典配乐/静默的力量 | 流行音乐/对白驱动 | 自然声/环境音/声音作为角色 |
| **典型节奏** | 缓慢但有目的性 | 快速剪辑+长段对话 | 极慢/冥想式/沉浸式 |
| **适合时长** | 长片（90-180分钟） | 长片（120-180分钟） | 长片（110-200分钟） |
| **AI生成难度** | ⭐⭐⭐（需维持对称和精密） | ⭐⭐⭐⭐（非线性结构复杂） | ⭐⭐⭐⭐⭐（长镜头和诗意难以用AI捕捉） |

### 📷 摄影 Cinematography (`cinematography/`)

**2种摄影美学，从自然光纪实到理论驱动的实验影像。**

| 技能路径 | 风格/理论 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|----------|-----------|------------|-----------|
| `cinematography/liao-fan-light` | 廖本榕 · 台湾新电影自然光美学 | 光是场景中最诚实的演员 | 文艺片/纪录片感/柔和氛围；追求自然光运用、台湾电影质感、纪实美学 | "自然光""廖本榕""台湾电影""纪实感""柔和""侯孝贤" |
| `cinematography/deleuze-time-image` | 吉尔·德勒兹 · 时间-影像理论 | 运动影像之外的时间表达 | 实验电影/艺术短片/意识流；追求打破因果逻辑、纯视听情境、时间自主性 | "德勒兹""时间影像""实验""意识流""理论""艺术电影" |

#### 摄影技能选择速查表

```
我想...                          → 推荐技能
─────────────────────────────────────────────
拍出温暖、自然、纪实感的画面      → liao-fan-light
做实验性的、打破常规的影像作品    → deleuze-time-image
```

**两种摄影美学的本质区别**

```
廖本榕（自然光美学）：
  - 目标：让光线自己"说话"
  - 方法：等待/观察/顺应自然光的变化
  - 效果：温暖、真实、有人间烟火气
  - 适用：剧情片/纪录片/Vlog

德勒兹（时间-影像）：
  - 目标：创造"脱离运动"的纯粹时间体验
  - 方法：打断叙事连贯性/使用非理性剪辑
  - 效果：梦幻、抽象、挑战观众认知
  - 适用：实验短片/MV/艺术装置
```

### 📱 短视频 Short Video (`short-video/`)

**2种短视频范式，从算法友好到生活叙事。**

| 技能路径 | 类型 | 一句话定位 | 最佳适用场景 | 关键触发词 |
|---------|------|-----------|------------|-----------|
| `short-video/douyin-trend` | 抖音爆款逻辑 | 3秒定生死的注意力工程 | 抖音/快手/小红书等平台；追求高完播率、快速传播、算法友好型内容 | "抖音""爆款""短视频""快节奏""算法""流量密码" |
| `short-video/vlog-storytelling` | Vlog 叙事法 | 生活本身就是一个好故事 | 个人IP/生活方式分享/旅行记录；追求真实感、个人魅力、叙事弧光 | "Vlog""生活记录""叙事""真实感""个人IP""日常" |

#### 短视频技能选择速查表

```
我想做...                        → 推荐技能
─────────────────────────────────────────────
一个能在社交平台快速传播的短片    → douyin-trend
记录我的日常生活并讲成故事        → vlog-storytelling
```

**两种短视频范式的关键差异**

| 维度 | 抖音爆款 | Vlog叙事 |
|------|---------|---------|
| **核心目标** | 传播最大化 / 算法优化 | 连接建立 / 社群沉淀 |
| **注意力策略** | 前3秒钩子 + 密集刺激 | 慢热建立信任 + 情感共鸣 |
| **时长偏好** | 15-60秒（黄金区间） | 3-10分钟（深度内容） |
| **剪辑节奏** | 极快（1.5-2秒一切） | 自然流畅（跟随叙事节奏） |
| **成功指标** | 完播率/点赞/转发/评论量 | 订阅转化/粉丝粘性/品牌合作 |
| **变现模式** | 流量分成/广告/带货 | 品牌赞助/知识付费/社群 |
| **生命周期** | 爆发性但短暂（热点过去就凉） | 长尾效应（优质Vlog持续有价值） |

## 选择决策树："我想做什么类型的视频？"

当用户说"帮我做一个视频"但不确定具体风格时，使用此决策树：

```
开始：用户的视频需求是什么？
│
├─ 时长判断
│  ├─ "< 1分钟 / 短平快"         → 进入 [短视频分支]
│  ├─ "1-5分钟 / 中等长度"        → 进入 [中篇分支]
│  └─ "> 5分钟 / 长片级"          → 进入 [长片分支]
│
├─ [短视频分支] （short-video/）
│  ├─ 追求传播/流量？             → douyin-trend (抖音爆款)
│  └─ 追求真实/连接？            → vlog-storytelling (Vlog)
│
├─ [中篇分支] （可能涉及多个板块）
│  ├─ 商业/产品介绍？             → douyin-trend (缩短为广告版)
│  ├─ 教学/教程？                 → vlog-storytelling (结构化版)
│  └─ 艺术/实验？                 → cinematography/deleuze-time-image
│
├─ [长片分支] （directing/ + cinematography/）
│  ├─ 题材类型？
│  │  ├─ 史诗/科幻/战争           → epic-kubrick (库布里克)
│  │  ├─ 犯罪/黑色幽默/后现代     → non-linear-tarantino (昆汀)
│  │  ├─ 艺术/哲学/内心世界       → tarkovsky-poetic (塔可夫斯基)
│  │  └─ 纪实/人文/温情           → cinematography/liao-fan-light (廖本榕)
│  │
│  └─ 视觉优先还是故事优先？
│     ├─ 视觉优先（画面即叙事）    → 库布里克 或 塔可夫斯基
│     └─ 故事优先（对白/结构）     → 昆汀 或 配合 writing/screenplay/
│
└─ [通用诊断] （5个快速问题）
   Q1: 受众是谁？
       → 大众/泛娱乐 → 抖音/昆汀
       → 文艺青年/专业圈 → 塔可夫斯基/德勒兹
       → 特定社群 → Vlog
   Q2: 平台在哪里发布？
       → 抖音/快手/TikTok → douyin-trend
       → B站/YouTube → 可选任何（取决于内容深度）
       → 朋友圈/私域 → Vlog/廖本榕风格
       → 艺术展/电影节 → 塔可夫斯基/德勒兹
   Q3: 想传达什么情绪？
       → 震撼/敬畏 → 库布里克
       → 兴奋/紧张 → 昆汀
       → 宁静/沉思 → 塔可夫斯基/廖本榕
       → 快乐/轻松 → 抖音/Vlog
   Q4: 资源约束？
       → 无专业设备/一人团队 → Vlog/抖音（手机即可）
       → 有一定设备/小团队 → 廖本榕/昆汀
       → 专业制作/充足预算 → 库布里克/塔可夫斯基
   Q5: AI参与程度？
       → 全AI生成（文生视频）→ 任何都行（AI目前擅长短片段）
       → AI辅助（实拍+AI特效）→ 库布里克/昆汀（视觉增强）
       → 实拍为主/AI辅助后期 → 廖本榕/Vlog（调色/剪辑）
```

## 与 AI 视频大师的协作流程

> **本领域的所有技能都需要配合 `ai-video-master` 才能完成从创意到成片的完整闭环。**
>
> **注意：当前AI视频生成技术（2025年水平）最适合短视频和中长片段。长片级作品建议采用"AI辅助实拍"模式。**

### 协作架构

```
用户需求："做一个塔可夫斯基风格的关于时间的艺术短片"

方案A：纯AI生成（推荐用于 < 60秒 片段）
│
├─ Step 1: 加载电影大师技能 (品味层)
│  └── directing/tarkovsky-poetic/SKILL.md
│      ├── 提供：塔可夫斯基的时间哲学
│      ├── 提供：感知训练（如何"观看"时间）
│      ├── 提供：长镜头美学关键词库
│      └── 提供：诗电影评价标准
│
├─ Step 2: 构建"品味+工艺"提示词
│  └── 结合：
│      ├── 大师的视听语言特征（来自Step 1）
│      └── AI视频平台的参数适配（见下方）
│
├─ Step 3: AI视频大师执行 (工艺层)
│  └── ai-video-master/SKILL.md
│      ├── 选择平台（Seedance 2.0? Runway Gen-3? Kling? Sora?）
│      ├── 将电影语法转化为提示词+参数
│      ├── 生成初稿片段（每段5-15秒）
│      └── 自动迭代优化（最多5轮）
│
└─ Step 4: 电影大师评价 (回到品味层)
   └── 使用塔可夫斯基的自检清单评分
       ├── 时间流动性 ✓
       ├── 诗意意象密度 ✓
       ├── 长镜头沉浸感 ✓（或AI近似效果）
       ├── 声画关系 ✓
       └── 精神维度 ✓
       → 输出最终视频 + 评价报告


方案B：AI辅助实拍（推荐用于 > 60秒 或高质量项目）
│
├─ Step 1: 加载目标技能（同上）
├─ Step 2: 用AI大师辅助前期
│  ├── 生成分镜脚本（基于大师风格）
│  ├── 生成参考图板（mood board）
│  └── AI预演（animatic）验证节奏
├─ Step 3: 实际拍摄
│  └── 按分镜执行，可实时调用AI建议
├─ Step 4: AI辅助后期
│  ├── AI调色（匹配大师色调）
│  ├── AI特效/合成（如需要）
│  └── AI辅助剪辑（粗剪→精剪）
└─ Step 5: 大师标准最终审查
```

### 不同平台的适配建议

| 目标平台 | 推荐搭配技能 | 当前能力上限 | 备注 |

## 典型工作流示例

### 示例1：新手首次使用 — 抖音短视频

```
用户："帮我想一个抖音爆款短视频的创意"

系统响应：
  1. 路由判断：[短视频] → [追求传播] → short-video/douyin-trend
  2. 加载 douyin-trend/SKILL.md
  3. 引导完成：
     - 感知热身（5分钟）：分析3个当前爆款视频的结构
     - 创意生成（使用"爆款公式"）：
       "[前3秒钩子] + [情绪峰值×3] + [反转/惊喜] + [CTA]"
     - 调用 ai-video-master 生成脚本 + 参考分镜
     - 用抖音自检清单评估：
       ✓ 前3秒是否有明确钩子？（✓ "你绝对想不到..."）
       ✓ 是否在15秒内有第一个情绪高峰？
       ✓ 结尾是否有引导互动的设计？
       ✓ 总时长是否控制在30-45秒（黄金区间）？
  4. 输出：完整脚本 + 分镜描述 + BGM建议 + 发布策略
```

### 示例2：熟练用户 — AI生成电影片段

```
用户："用库布里克的风格做一个太空站的镜头"

自动路由：
  → directing/epic-kubrick/SKILL.md (品味层)
  → ai-video-master/SKILL.md (工艺层)

执行流程：
  1. 加载库布里克的"单点透视构图法则"
  2. 构建提示词（以Runway为例）：
     "Symmetrical one-point perspective shot of a futuristic space station
     interior, wide angle lens, cold blue and white color palette,
     Stanley Kubrick aesthetic, 2001 A Space Odyssey influence,
     slow dolly-in movement, geometric precision, clinical lighting,
     empty corridor with vanishing point, 8k resolution,
     cinematic aspect ratio 2.39:1"
  3. AI大师执行：生成10秒片段
  4. 库布里克自检：
     ✓ 对称性是否严格？（✓ 中央构图，左右镜像）
     ✓ 单点透视是否清晰？（✓ 明确的消失点）
     ✓ 色调是否冷峻？（✓ 蓝/白主调，低饱和）
     ✓ 运镜是否有仪式感？（✓ 缓慢推进，像朝圣）
     ✓ 是否有"非人"的疏离感？（✓ 空旷无人，尺度压倒性）
  5. 输出：高质量电影片段 + 技术参数记录
```

### 示例3：跨技能融合（高级）

```
用户："做一个王家卫式的独白，但是用塔可夫斯基的长镜头拍摄"

融合模式：wong-mood-fragment(主风格) × tarkovsky-poetic(主技法)
  主风格：王家卫的情绪碎片（独白文本/时间标记/物象隐喻）
  主技法：塔可夫斯基的长镜头美学（深焦/横移/时间凝固）

执行方案（AI辅助实拍模式）：
  1. 从 writing/screenplay/wong-mood-fragment 获取独白脚本
  2. 从 directing/tarkovsky-poetic 获取镜头设计：
     - 使用深焦镜头（前景中景背景都清晰）
     - 设计90-120秒的单一横移镜头
     - 场景：雨后的香港街道（王家卫元素）
     - 但用塔可夫斯基的方式拍：缓慢、凝视、让环境"说话"
  3. AI辅助：
     - 生成分镜图板（结合两种风格参考）
     - AI预演验证节奏
     - 实拍后AI调色（冷暖对比：霓虹暖 vs 雨水冷）
  4. 最终效果预期：
     "一段王家卫式的独白，但不是用手持晃动镜头拍，
      而是用塔可夫斯基式的庄严长镜头来呈现。
```

## 重要提醒：视频创作的特殊性

相比于文字和图像，视频创作有以下独特挑战：

### 1. 时间维度的复杂性
视频是时空艺术，需要在**时间流**中控制信息释放节奏。这也是为什么本领域强调"时间感知训练"。

### 2. 技术门槛更高
即使有AI辅助，视频仍涉及：
- 分镜/故事板设计
- 镜头语言（景别/角度/运动）
- 声音设计（对白/音效/音乐）
- 剪辑节奏
- 色彩分级

**建议新手先从短视频开始，建立基本的视听语言感觉。**

### 3. AI视频的当前局限（2025年水平）
- **一致性**：长视频中人物/场景的一致性仍是挑战
- **物理真实性**：复杂的物理交互（液体/碰撞/柔软物体）容易出错
- **精确控制**：难以精确控制每一帧的细节（不像图像可以逐像素调整）
- **时长限制**：大多数平台仍限制在15秒以内的高质量输出

**应对策略**：
- 将长项目拆分为多个短片段分别生成
- 采用"AI生成B-roll + 实拍主线"的混合模式
- 把重点放在创意和策划上，而不是追求AI独立完成一切

---

> **记住：这个领域不是教你"怎么用工具剪视频"，而是教你怎么"像导演一样思考和观看"。**
> **技术会不断进步，但对时间和空间的感知力是永恒的创作基础。**
> **最好的AI视频工具是你的眼睛和你的心。**
