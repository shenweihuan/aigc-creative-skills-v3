## 一句话定位

**当你用 AI 创作时，这个系统帮你解决一个问题：生成的东西"能用但不惊艳"。**

不是教你按按钮——是教你在按下按钮之前，先建立**大师级的审美判断力**。

## 30秒上手（傻瓜式入口）

### 你想做什么？

| 你想做 | 第一步 | 耗时 |
|---------|--------|------|
| **画一张图** | 直接说 → 系统自动选风格大师 + AI绘画大师执行 | 5分钟出图 |
| **写一段文字** | 直接说 → 系统自动选文体大师 + AI写作大师执行 | 5分钟成稿 |
| **做一个视频** | 直接说 → 系统自动选导演风格 + AI视频大师执行 | 10分钟成片 |
| **设计一个东西** | 直接说 → 系统自动选设计流派 + 执行 | 5分钟出方案 |
| **不知道用什么风格** | 说"帮我推荐" → 风格诊断 → 匹配最适合你的大师 | 2分钟 |

### 具体怎么用？三个例子

**例1：你想画一张有氛围的图**
```
你只需要说：
"用莫奈的风格画一个秋天的湖边小屋"

系统会：
① 自动加载「莫奈·印象派」的品味框架（色彩/光影/笔触）
② 自动调用「AI绘画大师」选择平台（MJ/DALL-E/SD）
③ 用莫奈专属提示词公式构建 prompt
④ 生成 → 用莫奈五维标准自检 → 不满意就迭代

你不需要知道莫奈是谁，也不需要懂 MJ 参数。
```

**例2：你想写一段有力量的文字**
```
你只需要说：
"帮我用海明威的风格写一段关于加班的感受"

系统会：
① 加载海明威的冰山理论（说7分留93分不说破）
② 选择 Claude（最擅长文学模仿的平台）
③ 按海明威句式规则生成：短句、名词驱动、零情感词
④ 输出 + 六维评价（克制力/句健康度/冰深度...）

你得到的不是AI腔，是有骨头有节奏的文字。
```

**例3：你想做一个有质感的短视频**
```
你只需要说：
"做一个王家卫风格的雨夜城市短片，15秒"

系统会：
① 加载王家卫的情绪碎片叙事法（独白即剧情）
② 选择 Seedance 2.0（中文+国风最强）
③ 用王家卫式运镜（手持感/暧昧光/霓虹夜）
④ 生成 → 评价 → 迭代到满意为止
```

## 四种使用模式

| 模式 | 什么时候用 | 举例 |
|------|-----------|------|
| **🎓 教学** | 第一次用某个工具 | "教我用 Midjourney 画莫奈风格的画" |
| **🔍 评价** | 已经生成了东西，不知道好不好 | "帮我评价这张图" → 五维评分 + 具体修改方案 |
| **📋 工作流** | 要做一个完整项目 | "我要做一个系列海报" → 全流程规划 |
| **🔄 自动迭代** | 生成了但不够好 | "还不够好，再来一版" → 自动循环最多5轮直到满意 |

## 为什么不用只问 ChatGPT/Claude？

| | 只用 AI | 用 AIGC v3 系统 |
|---|--------|---------------|
| **风格** | 取决于 prompt 运气 | 由 34 位艺术大师保证 |
| **质量** | 不知道哪里不好 | 五维自检清单精确诊断 |
| **迭代** | 盲猜怎么改 | 自动迭代引擎定向优化 |
| **品味** | 永远无法训练 | 每个大师都有感知训练方法 |
| **一致性** | 每次结果风格飘忽 | 同一大师公式保证统一 |

## 从零到第一次输出的最快路径

```
Step 1（现在）：直接告诉我你想做什么
        例："用村上春树的风格写一段关于下雨的文字"

Step 2（1分钟后）：我会展示生成的结果 + 评分

Step 3（如果你愿意）：我们进入迭代模式
        → "还不够好" → 自动改进 → 再评 → 直到满意

Step 4（可选深入学习）：
        如果你对某个大师感兴趣，我可以带你走一遍他的感知训练
        → 这是让品味真正提升的关键步骤
```

**不需要读任何文档。不需要理解架构。现在就可以开始。**
